# BOSS v8.13.11

**Release Date**: December 2024

## Summary

This release focuses on authentication stability, UI improvements, and CLI enhancements with adaptive window sizing for laptop displays and standardized search functionality.

## Highlights

- Fixed duplicate window creation during magic link sign-in
- Adaptive window sizing optimized for laptop displays with scrollable UI
- Standardized search bars across the application
- CLI URL handling improvements

## Features

- **UI**: Implement adaptive window sizing for laptop displays with scrollable UI (#220)
  - Optimizes window dimensions for laptop screens
  - Adds scrollable containers for improved usability on smaller displays
  - Ensures UI remains accessible across different screen sizes

- **Search**: Standardize search bars and fix Secret Manager race condition (#221)
  - Unified search bar styling and behavior across the application
  - Fixed race condition in Secret Manager affecting reliability
  - Improved consistency in search user experience

## Bug Fixes

- **Auth**: Prevent duplicate windows during magic link sign-in (#222)
  - Fixed issue where multiple windows could open during authentication
  - Ensures clean authentication flow without window duplication
  - Improves user experience during sign-in process

- **CLI**: URL handling improvements (#219)
  - Enhanced CLI URL parsing and validation
  - Better error handling for malformed URLs
  - Improved command-line interface reliability
