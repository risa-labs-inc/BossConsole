# BOSS v8.12.3

**Release Date**: October 2024

## Summary

This release refactors the configuration system to workspaces with improved terminology and automatic cleanup of tabs when workspaces are deleted.

## Highlights

- Renamed configuration system to workspaces throughout the codebase
- Added automatic cleanup of tabs when a workspace is deleted
- Updated storage directory structure for better organization

## Improvements

### Workspace System Refactor (#36)

**Terminology Updates:**
- Renamed all "configuration" references to "workspaces" throughout the codebase
- Updated package structure from `components.configuration` to `components.workspaces`
- Renamed all classes, methods, and variables to use workspace terminology
- Updated UI text to display "Workspace" instead of "Configuration"
- Changed storage directory from `/configurations/` to `/workspaces/`

**Automatic Cleanup:**
- Added automatic cleanup of tabs when a workspace is deleted
- Implemented `cleanupDeletedWorkspace()` method in SplitView
- Added deletion callback system in WorkspaceManager
- Orphaned workspace states are now automatically cleaned up

**Breaking Changes:**
- All saved configurations will need to be re-saved as workspaces
- File paths have changed from `.../configurations` to `.../workspaces`
