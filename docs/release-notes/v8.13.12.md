# BOSS v8.13.12

**Release Date**: December 2024

## Summary

This release introduces Focus Mode, a productivity feature that allows users to minimize distractions by hiding UI panels through intuitive keyboard shortcuts.

## Highlights

- New Focus Mode with keyboard shortcuts for distraction-free work
- Hide/show panels with `Cmd/Ctrl+Shift+F` to maximize content area
- Seamless toggle between normal and focused workspace layouts

## Features

- **UI**: Implement Focus Mode with keyboard shortcuts (#74) (#227)
  - Added Focus Mode to hide side panels and maximize workspace
  - Keyboard shortcut: `Cmd+Shift+F` (macOS) / `Ctrl+Shift+F` (Windows/Linux)
  - Toggle between normal and focused layouts for distraction-free work
  - Improves productivity by maximizing content viewing area
