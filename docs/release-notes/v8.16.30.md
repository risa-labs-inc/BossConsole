# BOSS v8.16.30

**Release Date:** 2026-02-08

Plugin system enhancements with project selection API and minimum version validation for safer plugin installations.

## Highlights

- **Project Selection API**: New selectProject method in ProjectDataProvider enables codebase plugins to access project selector dialog
- **Version Validation**: Plugin loader now validates minBossVersion during installation to prevent incompatible plugins
- **Enhanced Plugin Safety**: Improved plugin compatibility checks ensure plugins work correctly with the current BOSS version

## Features

- **plugin-api**: Add selectProject to ProjectDataProvider for codebase plugin (#616)
  - Add selectProject() method to ProjectDataProvider interface for external codebase plugins
  - Allows plugins to prompt users to select projects from available list
  - Returns Project object with path and metadata
  - Bump plugin-api version to 1.0.14
  - Enables codebase-aware plugins to dynamically select target projects

- **plugin-loader**: Add minBossVersion validation during plugin installation (#615)
  - Validate minBossVersion from plugin manifest against current BOSS version
  - Throw PluginValidationException if plugin requires newer BOSS version
  - Display user-friendly error message with required vs current version
  - Prevents incompatible plugins from being installed
  - Improves plugin installation reliability and user experience
