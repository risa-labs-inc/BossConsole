# BOSS v8.15.2

**Release Date**: January 5, 2026

## Summary

This release improves window and panel management with fixes for per-window project isolation and panel mapping.

## Highlights

- Fixed per-window project isolation ensuring each window maintains independent project state
- Resolved panel mapping issues for better UI consistency

## Bug Fixes

### Window
- **Per-window project isolation and panel mapping fixes** (#433)
  - Fixed issues where project state could leak between windows
  - Improved panel slot mapping to ensure correct positioning across window instances (64f5697e)
