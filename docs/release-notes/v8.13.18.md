# BOSS v8.13.18

**Release Date**: December 2024

## Summary

This release fixes a browser URL bar regression where pressing Enter would navigate to the autocomplete suggestion instead of the user's typed text.

## Highlights

- Fixed URL bar Enter key behavior to use typed text instead of autocomplete suggestion
- Restored proper keyboard navigation: Enter for typed text, Tab for autocomplete

## Bug Fixes

- **Browser**: URL bar Enter key now uses typed text instead of autocomplete suggestion (#240)
  - Fixed regression where pressing Enter would navigate to autocomplete suggestion instead of user's typed text
  - Enter key now correctly uses what the user typed
  - Tab key continues to accept autocomplete suggestion (unchanged)
  - Arrow keys + Enter uses selected dropdown item (unchanged)
  - Fixes regression introduced in commit 1dc1524 (PR #226)
