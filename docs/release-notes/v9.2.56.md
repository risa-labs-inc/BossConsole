# BOSS v9.2.56

**Release Date:** 2026-07-23

Routes ~380 hardcoded color literals across ~35 UI files through the Operator's Console design system so every component honors live theme switching, and decomposes the monolithic `BossApp.kt` into focused effect files (#18).

## Highlights

- **Design-system colors everywhere**: ~380 hardcoded `Color(0x...)` / `Color.White`/`Black` literals across ~35 live UI files are replaced with semantic tokens (surfaces, borders, text, accents, status), so components now re-skin live when the theme changes instead of ignoring it (#18).
- **`BossApp.kt` decomposed**: the monolith is split into focused `app/` effect files, with `logger.warn` breadcrumbs added to seven previously silent `catch` blocks (#18).
- **Contrast & accessibility fixes**: Commit/Amend, Reset Browser, and other button labels now use `onSignal` on accent fills instead of low-contrast literals (#18).

## Bug Fixes

- **tabs**: Split menu no longer goes stale — split-menu enablement subscribes via `tabsState.subscribeAsState()` instead of reading a Decompose `Value.value` in composition (#18)
- **theme**: fix AWT background conversion in `BossWindow` — use `toArgb()` instead of `value.toInt()`, which read the empty low bits of the ARGB-packed `ULong` (#18)
- **theme**: raise button-label contrast — `CommitDialog` Commit/Amend and Fluck Reset Browser now render `onSignal` on accent fills instead of ~1.7:1 literals (#18)

## Improvements

- **theme**: route hardcoded colors through the design system with role-mapped semantic tokens; token reads are never cached so theme switches re-skin everything live (#18)
- **theme**: delete private mini-palettes (UpdateUI, wizard, GlobalSearchDialog) in favor of use-site token reads (#18)
- **editor**: extract the duplicated VS Code Dark+ syntax palette (5 copies) into one shared `EditorSyntaxColors` object (#18)
- **terminal**: derive the Swing find bar (FluckEngine) `java.awt` colors from the same tokens at build time (#18)
- **app**: decompose `BossApp.kt` into focused `app/` effect files and add `logger.warn` breadcrumbs to seven silently-swallowed catches (#18)
- **theme**: fix the stale "dark-only" KDoc on `BossTheme` — a user-selectable light Daylight theme ships (#18)
