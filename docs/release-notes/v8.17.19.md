# BOSS v8.17.19

**Release Date:** 2026-03-08

Bug fix release improving plugin binary compatibility validation by skipping intra-JAR references that are guaranteed to be self-consistent.

## Highlights

- **Intra-JAR validation false positives fixed**: The binary compatibility validator no longer flags references between classes within the same JAR, eliminating false positives for plugins like `boss-plugin-api` that bundle their own copies of shared classes

## Bug Fixes

- **plugins**: Skip intra-JAR references in binary compatibility validation — classes compiled into the same JAR are self-consistent; only cross-JAR references (plugin → host API) require validation
