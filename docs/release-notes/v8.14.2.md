# BOSS v8.14.2

**Release Date**: December 2025

## Summary

This release focuses on browser stability improvements with enhanced JavaScript dialog handling, file:// URL support, and better mouse button behavior across platforms.

## Highlights

- **BOSS-styled JavaScript dialogs** - Custom-styled alert/confirm/prompt dialogs matching BOSS design
- **File URL support** - Open local files directly in browser with file:// protocol
- **Mouse button handling** - Fixed middle-click and button navigation issues (#325, #328)
- **Split view improvements** - Prevent duplicate splits when using terminal links with "Remember" option

## Features

### Browser
- **BOSS-styled JavaScript dialogs** (#372)
  - Custom alert, confirm, and prompt dialogs with BOSS styling
  - Configurable settings for JavaScript dialog behavior
  - Warning messages for auto-confirmed dialogs
  - Prevents duplicate dialogs across multiple browser tabs

### Terminal
- **Terminal link split view fix** (#371)
  - Prevent duplicate split views when clicking terminal links with "Remember" option
  - Improved handling of remembered split preferences

## Bug Fixes

### Browser
- **JavaScript dialog freeze and file:// URL support** (#369, #372)
  - Resolved freeze issues when JavaScript dialogs appeared
  - Added support for file:// URLs in browser
  - Case-insensitive URL scheme detection
  - Debug logging for URL input processing

- **Mouse button handling** (#368, #325, #328)
  - Fixed mouse button handling for browser tabs
  - Added fallback support for Linux mouse button mapping
  - Improved middle-click and navigation button behavior

### UI
- **Update banner in focus mode** (#370)
  - Update banner now visible in focus mode
  - Ensures users see important update notifications
