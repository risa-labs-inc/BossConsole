# BOSS v8.17.18

**Release Date:** 2026-03-07

Stability improvements adding plugin binary compatibility validation at load time with fixes for a Compose layout reentrancy crash and an ineffective binary compatibility check.

## Highlights

- **Plugin binary compatibility validation**: Plugins are now validated against the host ABI at load time, catching `NoSuchMethodError` and similar binary incompatibilities before they can crash the app
- **Compose layout reentrancy crash fix**: Resolved the "layout state is not idle" crash caused by triggering layout passes during an active layout phase
- **Validator correctly catches NoSuchMethodError**: The binary compatibility validator was not actually intercepting `NoSuchMethodError`; this has been corrected

## Features

- **plugins**: Add plugin binary compatibility validation at load time

## Bug Fixes

- **plugins**: Fix binary compatibility validator to actually catch `NoSuchMethodError`
- **ui**: Fix Compose layout reentrancy crash ("layout state is not idle")

## Improvements

- **ui**: Document popup position staleness trade-off in layout refs
