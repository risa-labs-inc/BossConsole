# BOSS v8.12.23

**Release Date**: October 2024

## Summary

This release resolves two critical browser stability issues: SIGSEGV crashes when closing windows and unwanted automatic window recreation on macOS.

## Highlights

- Fixed JxBrowser SIGSEGV crashes caused by uninitialized browser frames during window close
- Resolved auto-window creation bug that violated standard macOS window lifecycle behavior
- Improved browser initialization with lazy loading to prevent UI thread blocking
- Enhanced thread safety with proper disposal guards and environment validation

## Bug Fixes

### Browser
- **Fixed JxBrowser SIGSEGV crashes and auto-window creation bug** (#121)
  - Added window validation in `createBrowserViewState()` to only use displayable and showing windows
  - Implemented lazy browser initialization via `LaunchedEffect` to prevent UI thread blocking
  - Enhanced disposal guards with `isBrowserEnvironmentValid()` checks across all browser coroutines
  - Removed Thread.sleep() anti-patterns for cleaner async handling
  - Fixed macOS window lifecycle: closing last window now keeps app running in Dock (standard macOS behavior)
  - Moved initial window creation before `application{}` reactive block to prevent unwanted auto-recreation
  - Eliminated race conditions during window close and browser disposal
