# BOSS v8.13.14

**Release Date**: December 2024

## Summary

This release enhances browser usability with favicon support in navigation UI, improved URL bar behavior, faster link responsiveness, and streamlined authentication infrastructure.

## Highlights

- Added favicon display in quick switcher and tab buttons for better visual navigation
- Fixed browser link click delay by switching to LoadStarted event detection
- Resolved URL bar text overlapping issue during typing
- Completed removal of local passkey storage management infrastructure

## Features

- **Browser**: Display favicons in quick switcher and tab buttons (#224) (#230)
  - Enhanced visual navigation with favicon support
  - Improved tab and quick switcher UI consistency

## Bug Fixes

- **Browser**: Eliminate link click delay by using LoadStarted event (#229) (#232)
  - Switched from LoadFinished to LoadStarted event for instant link feedback
  - Significantly improved browser responsiveness and user experience

- **Browser**: Resolve text overlapping in URL bar during typing (#223) (#231)
  - Fixed visual glitch where typed text overlapped with existing content
  - Improved URL bar text rendering and editing experience

## Improvements

- **Auth**: Complete removal of local passkey storage management (#228)
  - Streamlined authentication infrastructure
  - Removed obsolete local storage code paths
