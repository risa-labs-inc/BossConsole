# BOSS v8.16.20

**Release Date:** 2026-01-30

Stable release featuring critical CodeBase panel fixes and enhanced CI/CD workflow capabilities.

## Highlights

- **CodeBase Panel Fix**: Restored 180+ file icons in CodeBase panel for proper file type visualization
- **Enhanced Release Workflow**: Stage-by-stage promotion support with dynamic pre-release selection
- **Improved Error Handling**: Better input validation with helpful error messages in promotion workflow

## Bug Fixes

- **codebase**: Restore 180+ file icons in CodeBase panel ([#574](https://github.com/risa-labs-inc/BossConsole/pull/574))
- **workflow**: Revert to string input with available options on error

## Features

- **workflow**: Consolidate promote-release with dynamic dropdown
- **workflow**: Add list-prereleases helper workflow
- **workflow**: Show available pre-releases on invalid input
- **workflow**: Add stage-by-stage promotion support
