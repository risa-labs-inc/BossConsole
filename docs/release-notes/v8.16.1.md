# Release Notes - v8.16.1

**Release Date**: January 23, 2026

This patch release upgrades the integrated browser engine to JxBrowser 8.16.1 (Chromium 144) for improved performance and stability.

## Highlights

- Updated JxBrowser from 8.15.0 to 8.16.1 (Chromium 144)
- Centralized version management in gradle/libs.versions.toml
- Fixed branded Chromium workflow version handling

## Dependencies

- **browser**: Upgrade JxBrowser from 8.15.0 to 8.16.1 (#514)
  - Updated to JxBrowser 8.16.1 (Chromium 144) for improved browser performance and stability
  - Centralized JxBrowser version definition in gradle/libs.versions.toml as single source of truth
  - Generated JXBROWSER_VERSION constant in VersionConstants.kt from TOML for runtime access
  - Updated ChromiumAutoDownloader to use generated constant
  - Added -PjxBrowserVersion Gradle property for CI build version overrides
  - Fixed branded Chromium workflow to correctly pass version to Gradle task
