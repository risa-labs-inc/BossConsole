# BOSS v8.13.19

**Release Date**: December 2024

## Summary

This release enables Gradle configuration cache support for significantly faster builds, improving developer experience with 10-50% build time reduction.

## Highlights

- Gradle configuration cache compatibility enabled for faster builds
- Proper task input/output tracking for incremental builds
- Refactored native library extraction tasks for build system stability

## Bug Fixes

- **Build System**: Enable Gradle configuration cache compatibility (#244)
  - Refactored extractPty4jNative and extractJcefNatives tasks to support configuration cache
  - Moved layout.buildDirectory and project.configurations access to configuration time using providers
  - Replaced Gradle copy DSL with Java ZipFile API to avoid script context serialization issues
  - Added proper inputs/outputs declarations for accurate up-to-date checking
  - Configuration cache can now be enabled with --configuration-cache flag (10-50% faster builds)
  - Cache entries are stored and reused on subsequent builds
  - No functional changes to native library extraction behavior
