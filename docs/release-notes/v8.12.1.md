# BOSS v8.12.1

**Release Date**: October 2024

## Summary

This release addresses critical macOS code signing and notarization issues to ensure proper distribution and security compliance for macOS builds.

## Highlights

- Fixed macOS app bundle code signing after PTY4J modifications for proper notarization
- Enhanced CI/CD pipeline with detailed notarization error logging
- Improved macOS build reliability and debugging capabilities

## Bug Fixes

### macOS Code Signing and Notarization

**Code Signing Fix:**
- Re-sign app bundle after PTY4J modifications to meet Apple's notarization requirements
- Ensures proper code signing of all native components and frameworks
- Resolves notarization failures caused by unsigned or modified binaries

**CI/CD Improvements:**
- Added detailed notarization error logging to macOS build workflow
- Improved debugging capabilities for code signing and notarization issues
- Enhanced visibility into Apple's notarization validation process
