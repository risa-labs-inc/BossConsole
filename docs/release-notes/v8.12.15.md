# BOSS v8.12.15

**Release Date**: October 2024

## Summary

This release improves authentication user experience by providing clear feedback when magic links expire or have already been used, eliminating silent failures during the login process.

## Highlights

- Added user feedback for expired and already-used magic links
- Clear error messages displayed in the authentication UI
- Automatic error clearing on navigation

## Bug Fixes

### Authentication
- **User feedback for expired/old magic links** (#103, Issue #101)
  - Fixed silent failures when clicking expired or already-used magic links
  - Clear error messages now displayed above "Resend Magic Link" button
  - Example messages:
    - "This magic link has expired. Magic links are valid for 15 minutes..."
    - "This magic link has already been used..."
    - "Network request cancelled. Please check your internet connection."
  - Errors automatically clear when navigating away
