# BOSS v8.16.10 Release Notes

**Release Date:** 2026-01-25

## Summary

This release adds Git repository cloning with authentication support, improves crash handling by showing crash dialogs in separate windows, and includes comprehensive error handling for Git operations.

## Highlights

- Git clone repository functionality with SSH agent and credential helper support
- Crash dialogs now render in separate windows for UI thread crashes
- Step-by-step wizard UI for cloning repositories with real-time progress
- Enhanced error handling with timeouts for Git operations

## Features

### Git Integration
- **Add clone repository with authentication and window selection** (#550) (#556)
  - Clone repositories with proper SSH agent access via SSH_AUTH_SOCK
  - Support for git credential helpers for HTTPS authentication
  - Step-by-step wizard UI with URL validation and directory conflict detection
  - Auto-derive directory name from repository URL
  - Real-time progress updates during clone operation
  - Window selection dialog: open in current window or new window
  - Environment variable inheritance in ProcessBuilder for all git operations
  - 10-minute timeout mechanism to prevent indefinite hangs
  - Comprehensive error handling with pattern matching for IOException, SecurityException, and TimeoutCancellationException
  - Automatic cleanup of partial clones on timeout or interruption

## Bug Fixes

### Crash Handling
- **Show crash dialog in separate window for UI thread crashes** (#543) (#557)
  - Fix crash dialog not appearing when main Compose window breaks
  - Render crash dialog in standalone AWT/Swing JFrame with ComposePanel
  - Add global uncaught exception handler via CrashHandler
  - Include CrashReportService for GitHub issue submission
  - Add CrashReport data classes and CrashSignature for deduplication
  - Sanitization helpers in LogSanitizer for secure crash reports
  - "Trigger Test Crash" menu item for testing crash handling
