# BOSS v8.17.3

**Release Date:** 2026-02-16

Critical macOS build fix ensuring proper Chromium binary extraction with preserved symlinks for stable browser functionality.

## Highlights

- **macOS Build Fix**: Fixed Chromium binary extraction on macOS using `ditto` to properly preserve symlinks and prevent browser runtime issues

## Bug Fixes

- **browser**: Use macOS ditto for Chromium zip extraction to preserve symlinks (#631)
  - Replaced `unzip` with `ditto -xk` on macOS for extracting BOSS-branded Chromium
  - Ensures symlinks are properly preserved during extraction
  - Prevents potential browser initialization failures due to broken symlinks
  - Maintains existing `unzip` behavior for Windows and Linux builds
