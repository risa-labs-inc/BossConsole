# BOSS v8.13.0

**Release Date**: November 2025

## Summary

This major release introduces the comprehensive Bookmarks Plugin, providing powerful tab and workspace management capabilities with favorites, collections, search, and multi-workspace targeting.

## Highlights

- Complete Bookmarks Plugin with sidebar panel for managing tabs and workspaces
- Bookmark tabs to collections with right-click context menus
- Favorite workspaces for quick access
- Search and filter bookmarks in real-time
- Multi-workspace targeting: open bookmarks in specific workspaces and panels
- Expandable workspace views showing all contained tabs
- Create and manage bookmark collections and workspaces from the plugin
- Comprehensive context menus throughout the plugin

## Features

### Bookmarks Plugin (#146, #170, #178)
- **Data Layer**
  - Persistent bookmark storage in `~/Documents/BOSS/bookmarks/`
  - Support for browser, editor, and terminal tab bookmarks
  - Auto-creates "Favorites" collection on first run
  - Thread-safe coroutine-based operations with reactive StateFlows

- **Plugin UI**
  - Four collapsible sections: Favorites, Collections, All Workspaces, Favorite Workspaces
  - Material Design icons throughout (replaced emojis)
  - Display cached favicons for browser bookmarks
  - Real-time search bar with filtering across all sections
  - Expandable workspaces showing nested split structure and tabs
  - Visual split section dividers (Left/Right, Top/Bottom)

- **Tab Bookmarking**
  - Right-click any tab → "Add to Bookmarks"
  - Select target collection via BookmarkDialog
  - Multi-select collections and workspaces for bookmark creation
  - Choose specific panel/split within target workspace
  - Star icons on bookmarks for quick removal
  - Bookmark workspace tabs directly from expandable view

- **Workspace Favorites**
  - Multiple ways to favorite workspaces:
    - Tab context menu
    - Empty tab bar space context menu
    - Inline star button in Bookmarks plugin
    - Workspace context menu
  - Gold filled star for favorited, gray outlined star for unfavorited
  - Favorites persist across sessions

- **Collection Management**
  - Create new collections via plus icon
  - Rename collections with validation
  - Delete collections (except Favorites)
  - Copy bookmarks to multiple collections
  - Move bookmarks between collections
  - Clear all favorites with confirmation

- **Workspace Management**
  - Create new workspaces from plugin
  - Rename workspaces with validation
  - Delete workspaces (except predefined ones)
  - Export workspace as JSON
  - Unfavorite all workspaces with confirmation
  - Load entire workspace or open individual tabs
  - Click workspace name to load, click tab to open just that tab

- **Multi-Workspace Targeting**
  - Specify target workspace and panel when bookmarking
  - BookmarkDialog with horizontal 3-column layout
  - Multi-select workspaces with inline panel dropdowns
  - One bookmark can open in multiple workspace-panel pairs
  - Pills show selected panels (e.g., "work: Top Panel")

- **Context Menus**
  - BookmarkItem: Remove, Copy to Collection, Move to Collection, Add to Workspace, Edit Targets
  - CollectionItem: Rename, Export (TODO), Delete
  - WorkspaceItem: Load, Favorite/Unfavorite, Rename, Export, Duplicate (TODO), Delete
  - Section Headers: New Collection, Import Collection (TODO), New Workspace, Import Workspace (TODO)
  - Confirmation dialogs for all destructive actions

- **Integration**
  - Registered in DefaultPlugin with priority 1 (first position in sidebar)
  - Real-time UI updates when saving workspaces (fixed global singleton usage)
  - Force new tab creation when clicking bookmarks
  - Synchronized favorite state across all UI elements

## Improvements

### Bookmarks Plugin
- Improved WorkspaceManager to trigger immediate UI updates on save
- BookmarkDialog with expandable workspace section (collapsed by default)
- Increased empty tab bar space clickable area for context menu
- Only Favorites section expanded by default for cleaner initial view
- Chevron icons on workspace names to indicate clickability
- Proper theme colors (BossDarkSurface, BossDarkBorder) throughout UI
- Dynamic indentation for nested workspace tabs
- Reusable dialog components: ConfirmationDialog, RenameDialog, NameInputDialog
- CollectionSelectionDialog with multi-select (copy) and single-select (move) modes
- WorkspaceSelectionDialog with workspace-panel association
