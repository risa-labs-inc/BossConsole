# BOSS v8.16.34

**Release Date:** 2026-02-12

Critical terminal tab fix ensuring terminals open in the correct project directory.

## Highlights

- **Terminal Tab Directory Fix**: Fixed terminal tabs always opening with null working directory instead of project directory

## Bug Fixes

- **terminal**: Make TerminalTabInfo implement TerminalTabInfoInterface (d8c3e013)
  - TerminalTabInfo now properly implements TerminalTabInfoInterface
  - Fixed type casting issue in external terminal-tab plugin
  - Terminal tabs now correctly open in the project working directory instead of null
  - Resolved workingDirectory always being null due to missing interface implementation
