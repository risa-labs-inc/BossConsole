# BOSS v8.16.23-alpha.1

**Release Date:** 2026-02-05

Alpha release focused on dynamic plugin architecture with Maven Central publishing and runtime plugin management.

## Highlights

- **Dynamic Plugin Loading**: Plugins now load dynamically at runtime instead of being bundled with the application
- **Maven Central Publishing**: Plugin API published to Maven Central for public consumption
- **Runtime Plugin Management**: Install and manage plugins without restarting the application
- **Plugin Persistence**: Plugins persist across application sessions with automatic restoration

## Features

- **plugins**: Remove bundled panel plugins in favor of dynamic plugins (#603)
- **plugins**: Plugin persistence, API improvements, and auto-publish workflow (#602)
- **plugins**: Add plugin persistence and UI improvements (#587)
- **plugins**: Add runtime plugin installation and management (#544) (#578)
- **plugins**: Add Maven Central publishing support (#582)
- **plugins**: Change package group to ai.rever.boss.api for public release (#581)

## Bug Fixes

- **plugin-loader**: Use getURLs() method instead of urls property (#586)
- **plugins**: Apply maven-publish plugin with apply false at root (#584)
- **plugins**: Use vanniktech maven-publish plugin for Central Portal (#583)

## Improvements

- **workflow**: Remove old GitHub Packages publish workflow (#585)

## Dependencies

- **deps**: Combined dependency updates (#604)
