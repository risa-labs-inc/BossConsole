# BOSS v8.15.10

**Release Date**: January 11, 2026

## Summary

This release fixes branding inconsistencies in BOSS-branded Chromium notifications and adds Linux/FreeBSD support for terminal native library extraction.

## Highlights

- Fixed notification icon showing Chrome logo instead of BOSS logo
- Added Linux and FreeBSD support to pty4j native library extraction
- Fixed IPC race condition in Chromium binary extraction
- Improved notification system reliability

## Bug Fixes

### Chromium
- **Replace notification icon with BOSS logo** (#466, #468)
  - Fixed notifications showing Chrome icon instead of BOSS branding
  - Ensures consistent brand identity across all Chromium UI elements
  - Updated notification icon resource path

- **Fix notification icon showing Chrome instead of BOSS** (#470)
  - Additional fix for notification icon branding inconsistency
  - Improved icon resource loading mechanism

- **Add delay after extraction to avoid IPC race condition** (#469)
  - Fixed race condition in Chromium binary extraction process
  - Added proper delay to ensure IPC initialization completes
  - Prevents startup failures on first-run after Chromium download

### Terminal
- **Add Linux/FreeBSD support to pty4j native library extraction** (#471)
  - Extended native library extraction to support Linux and FreeBSD platforms
  - Ensures terminal functionality works correctly on all supported Unix-like systems
  - Improved cross-platform compatibility
