# BOSS v8.15.4

**Release Date**: January 5, 2026

## Summary

This release reverts a browser URL recovery fix due to stability concerns and updates several text editor dependencies for improved performance and compatibility.

## Highlights

- Reverted browser URL recovery fix (#444) to ensure browser stability
- Updated RSyntaxTextArea, AutoComplete, and RSTAUI libraries for better text editor performance

## Bug Fixes

### Browser
- **Reverted URL recovery fix** (#444)
  - Reverted previous fix for URL recovery using stale object reference (#434)
  - Ensures browser stability and prevents potential issues (47d4d053)

## Dependencies

### Text Editor Components
- **Bumped com.fifesoft:rsyntaxtextarea** (#441)
  - Updated to latest version for improved syntax highlighting (80ec22c3)

- **Bumped com.fifesoft:autocomplete** (#442)
  - Enhanced code completion capabilities (d7777edf)

- **Bumped com.fifesoft:rstaui from 3.3.1 to 3.3.2** (#440)
  - Latest UI improvements for text editor components (dc98712b)

## Improvements

### Dashboard
- **Improved browse suggestions ranking and fixed file card text clipping** (#435)
  - Better relevance in file suggestions
  - Fixed text display issues in file cards (849c3217)
