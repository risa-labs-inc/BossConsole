# BOSS v8.13.23

**Release Date**: December 2024

## Summary

This maintenance release focuses on dependency updates, including major version upgrades to Clikt 5.0 and JUnit 6.0, along with updates to Kotlin, Ktor, Supabase, and other core libraries.

## Highlights

- Upgraded Clikt CLI framework from 4.2.1 to 5.0.3 with API migration
- Updated JUnit testing framework from 5.10.0 to 6.0.1
- Bumped Kotlin from 2.3.0-Beta2 to 2.3.0-RC
- Updated Ktor from 3.3.1 to 3.3.2

## Dependencies

### Major Updates
- **Clikt CLI**: 4.2.1 → 5.0.3 (#277)
  - Migrated to new Clikt 5.0 API
  - Moved 'help' parameter from constructor to override function
  - Updated all CLI command classes to new API structure

- **JUnit Jupiter**: 5.10.0 → 6.0.1 (#277)

### Minor Updates
- **Kotlin**: 2.3.0-Beta2 → 2.3.0-RC (#281)
- **Ktor**: 3.3.1 → 3.3.2 (#277)
- **Supabase**: 3.2.5 → 3.2.6 (#277)
- **AndroidX Lifecycle**: 2.9.5 → 2.9.6 (#277)
- **ZXing**: 3.5.3 → 3.5.4 (#281)
- **Logback**: 1.5.20 → 1.5.21 (#281)
