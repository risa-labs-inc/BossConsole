# BOSS v9.2.28

**Release Date:** 2026-07-03

Maintenance release with no functional code changes — a version bump on top of v9.2.27 to publish a fresh signed multi-platform build. All v9.2.27 functionality carries forward unchanged.

## Highlights

- **Republished signed build**: A clean multi-platform build cut on top of v9.2.27 with no source changes; the only in-range commit is documentation (the v9.2.27 release notes).
- **v9.2.27 fixes carry forward**: The Toolbox 1.8.4 stability work — plugin updates gated on `minBossVersion`, `McpServerController.setPort` in the host interface, sequenced startup plugin passes, and best-effort `.deb` desktop-menu integration for headless installs — all ship unchanged.

## Changes

No functional code changes in this release. The only commit in the `v9.2.27..v9.2.28` range is documentation (adding the v9.2.27 release notes); version-bump commits are excluded.
