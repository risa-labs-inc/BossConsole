# BOSS v9.2.14

**Release Date:** 2026-06-27

Maintenance release with no functional code changes — a version bump on top of v9.2.13 to publish a fresh signed build. All v9.2.13 functionality (the granular RBAC role hierarchy, the `WindowManager` fit/restore API, and the email-link redirect fix) carries forward unchanged.

## Highlights

- **Republished build**: Re-cuts the signed multi-platform artifacts on top of v9.2.13 with no behavioral changes.

## Changes

No application code changed between v9.2.13 and v9.2.14. The only commits in range are the version bump and the v9.2.13 release-notes documentation.

For the feature set carried by this build, see the [v9.2.13 release notes](v9.2.13.md):

- Granular RBAC role hierarchy with permission-based plugin gating (#793)
- `WindowManager` fit/restore window-resize API for embedded BossTerm panes (#779)
- Public email-link auth redirect fix (`verify_jwt=false`) (#787, #791)
