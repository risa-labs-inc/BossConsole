# BOSS v8.15.5

**Release Date**: January 6, 2026

## Summary

This release introduces a new project creation wizard with templates, enhanced project management with better handling of deleted projects, and a convenient terminal context menu for accessing AI coding assistants.

## Highlights

- New project creation wizard with templates for quick setup
- Improved handling of deleted projects with auto-removal and notifications
- Terminal context menu for quick access to AI coding assistants
- Better workspace and project selection synchronization

## Features

### Project Management
- **Project creation wizard with templates** (#449)
  - Added guided wizard for creating new projects
  - Includes pre-configured templates for common project types
  - Streamlined project setup workflow (d640aa22)

- **Deleted project handling**
  - Display dialog when clicking on a deleted project (7e4bc40a)
  - Auto-remove deleted projects from recent list (2e2dbef0)
  - Added remove button to recent projects dropdown (519b8966)

### Terminal
- **Context menu for AI coding assistants** (#450)
  - Added right-click context menu in terminal
  - Quick access to popular AI coding tools
  - Enhanced terminal productivity workflow (d6d8ea57)

## Bug Fixes

### Project Management
- **Fixed workspace overwriting project selection** (#453)
  - Resolved issue where workspace changes would override selected project
  - Improved project state management (e53cbd75)

- **Improved project state synchronization**
  - Update global ProjectState when window selects project
  - Better consistency across application windows (c01742fd)

## Improvements

### Workflow
- Use PROJECT_TOKEN for add-to-project workflow (a83bc732)
- Auto-add issues to project board via GitHub Action (bb08bf09)
