# Release Notes - v8.15.26

**Release Date**: January 22, 2026

This release strengthens multi-window support by making event buses window-aware across the entire application, ensuring events are properly scoped to their source windows.

## Highlights

- Comprehensive event bus architecture update with sourceWindowId support
- All 11 event buses now properly handle multi-window scenarios
- Stricter event routing prevents cross-window event leakage

## Bug Fixes

- **eventbus**: Add sourceWindowId to all event buses for multi-window support (#506) (#507)
  - Made sourceWindowId required (not optional) for stricter event handling
  - Updated 11 event bus types: FileEventBus, TerminalEventBus, URLEventBus, PanelEventBus, DashboardEventBus (9 event types), RunEventBus, NavigationTargetBus, WorkspaceEventBus, RunnerTerminalEventBus, and TerminalLinkEventBus
  - CLI/external handlers now use WindowFocusManager for proper window targeting
  - Simplified subscriber filters by removing null-check fallback logic
  - Added null guards in composables using LocalWindowId.current
