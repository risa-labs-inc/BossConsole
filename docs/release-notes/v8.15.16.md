# BOSS v8.15.16

**Release Date**: January 11, 2026

## Summary

This release fixes critical macOS ARM64 stability issues and improves CI/CD reliability with enhanced release workflow automation.

## Highlights

- Fixed critical JVM crash on macOS ARM64 by disabling C2 JIT compiler
- Improved release workflow reliability with git synchronization
- Enhanced GitHub Actions compatibility across Windows ARM64 builds

## Bug Fixes

### JVM
- **Disable C2 JIT to prevent crash on macOS ARM64** (#476)
  - Added `-XX:TieredStopAtLevel=3` JVM flag to prevent C2 JIT compiler crashes
  - Stops compilation at tier 3 (C1 full optimization) instead of tier 4 (C2)
  - Resolves critical stability issues affecting macOS Apple Silicon users

### CI/CD
- **Fix Windows PowerShell syntax for ARM64 job**
  - Corrected PowerShell command syntax for environment variable handling
  - Improves build reliability on Windows ARM64 runners

- **Use Add-Content for GITHUB_ENV in Windows PowerShell**
  - Switched to `Add-Content` cmdlet for proper environment variable persistence
  - Ensures consistent behavior across PowerShell sessions

- **Use 'powershell' instead of 'pwsh' for Windows ARM64**
  - Changed to Windows PowerShell for better compatibility
  - Resolves script execution issues on ARM64 runners

- **Remove macOS x64 build (macos-13 retired)**
  - Removed macOS x64 build configuration as GitHub has retired macos-13 runners
  - Streamlined CI/CD pipeline to focus on supported platforms

- **Convert Windows ARM64 job to PowerShell**
  - Standardized Windows ARM64 builds to use PowerShell
  - Improved script consistency and maintainability

- **Add git pull --rebase before pushing version bump**
  - Added automatic rebase to prevent version bump conflicts
  - Ensures clean version bumps in release workflow

### Release Process
- **Add macOS architecture-specific DMGs and fix JDK distributions**
  - Separated macOS builds into ARM64 and x64 specific DMGs
  - Fixed JDK distribution selection for each architecture
  - Improved platform-specific build artifact naming
