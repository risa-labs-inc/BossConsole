# BOSS v8.16.26

**Release Date:** 2026-02-06

Enhanced plugin wizard with automatic triggering when no plugins are installed and improved modal dialog experience.

## Highlights

- **Auto-Trigger Wizard**: Plugin Setup Wizard automatically launches when no plugins are installed
- **Dialog Window Conversion**: Wizard now uses DialogWindow for better modal behavior and user experience
- **First-Time Setup**: Streamlined onboarding for new users with automatic plugin installation prompt

## Features

- **wizard**: Trigger plugin wizard when no plugins installed and convert to DialogWindow (#607)
  - Automatically show Plugin Setup Wizard when user has no plugins installed
  - Convert wizard from inline content to DialogWindow for proper modal behavior
  - Add UserDataStorage.hasNoPlugins() helper function to detect empty plugin state
  - Improve wizard UI presentation with dedicated dialog window
  - Ensure wizard only triggers once per session when no plugins are found
