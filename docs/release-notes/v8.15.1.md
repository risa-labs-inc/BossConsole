# BOSS v8.15.1

**Release Date**: January 5, 2026

## Summary

This release adds a new placeholder for conditional Claude Code continuation flags in workspace configurations.

## Highlights

- Added `{claudeContinueFlag}` placeholder for flexible workspace command configuration

## Features

### Workspace
- **Add {claudeContinueFlag} placeholder for conditional --continue** (#432)
  - Enables conditional use of the `--continue` flag in workspace configurations
  - Allows more flexible command templating for Claude Code integration (57c215ac)
