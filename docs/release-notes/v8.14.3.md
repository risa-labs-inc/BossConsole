# BOSS v8.14.3

**Release Date**: December 2025

## Summary

This release enhances terminal link handling with a new "Existing Split" option, allowing terminal hyperlinks to open in specific target splits, improving workflow flexibility and tab management.

## Highlights

- **Existing Split option for terminal links** - Open links in specific split locations (current, top, bottom, left, right)
- **Enhanced terminal link settings** - Configure default behavior for link clicks in terminal
- **BossTerm 1.0.64 update** - Latest improvements from BossTerm library
- **Improved edge case handling** - Better error handling and cleanup for terminal link operations

## Features

### Terminal
- **Add 'Existing Split' option for terminal link clicks** (#376)
  - Configure links to open in existing split locations
  - Target mode selection: current split, top, bottom, left, or right
  - Settings UI for default terminal link behavior
  - Improved workflow for opening files from terminal output

## Bug Fixes

### Terminal
- **Fix edge cases in terminal link handling**
  - Better error handling for invalid split targets
  - Improved cleanup on failed link operations
  - Code review feedback addressed for robustness

## Improvements

### Code Quality
- **Refactor terminal link feature**
  - Addressed code review feedback
  - Improved code organization and maintainability
  - Better separation of concerns for split target handling

## Dependencies

- **Update BossTerm to 1.0.64** (#375)
  - Latest bug fixes and improvements from BossTerm library
