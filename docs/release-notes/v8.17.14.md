# BOSS v8.17.14

**Release Date:** 2026-03-07

Critical bug fix ensuring the `SupabaseDataProvider` introduced in v8.17.13 is correctly accessible to sandboxed plugins via `SandboxedPluginContext`.

## Highlights

- **Sandboxed plugin Supabase access restored**: `SandboxedPluginContext` was missing the `supabaseDataProvider` delegation, causing all sandboxed plugins to receive `null` when attempting to use the new Supabase data API

## Bug Fixes

- **plugin**: Fix missing `supabaseDataProvider` delegation in `SandboxedPluginContext` — sandboxed plugins now correctly receive the underlying context's `SupabaseDataProvider` instead of `null`
