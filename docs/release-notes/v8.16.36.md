# BOSS v8.16.36

**Release Date:** 2026-02-12

Terminal plugin architecture refactoring with hot-reload support for improved modularity and developer experience.

## Highlights

- **Terminal Plugin Architecture**: Terminal functionality moved to plugin system for better modularity
- **Hot Plugin Reload**: Reload plugins without restarting the application via Help menu or panel context menu

## Features

- **terminal**: Move terminal to plugin and add plugin reload (#621)
  - Refactored built-in terminal into standalone `terminal-tab` plugin
  - Terminal logic now accessed via `TerminalAPIAccess` and `TerminalTabPluginAPI` interfaces
  - Removed direct `bossterm.compose` dependency from `composeApp` module
  - Added `Reload All Plugins` to Help menu for hot-reloading all plugins
  - Added `Reload Plugin` to sidebar panel context menus for individual plugin reload
  - Plugin reload implemented in `DynamicPluginManager` with reverse lookup support
  - `runtimeOnly` dependency strategy maintains compile-time decoupling while ensuring runtime availability
  - Graceful degradation with no-op methods when terminal plugin isn't loaded
