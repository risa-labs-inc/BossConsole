# BOSS v8.16.43

**Release Date:** 2026-02-12

Critical plugin memory leak fix ensuring proper cleanup of dynamic plugin browser instances when windows are closed.

## Highlights

- **Plugin Browser Cleanup**: Fixed memory leak where dynamic plugin browser instances were not disposed when closing windows

## Bug Fixes

- **browser**: Dispose dynamic plugin browsers on window close
  - Ensures DynamicPluginBrowser instances are properly cleaned up when closing a window
  - Prevents memory leaks from orphaned JxBrowser instances
  - Properly releases resources associated with plugin-created browser tabs
