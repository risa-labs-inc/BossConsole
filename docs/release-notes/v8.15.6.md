# BOSS v8.15.6

**Release Date**: January 6, 2026

## Summary

This release introduces powerful AI integration features for the terminal, including the BossTerm AI Assistant and a convenient context menu for quick access to popular AI coding assistants.

## Highlights

- Integrated BossTerm AI Assistant with Welcome Wizard support
- Added context menu in terminal for quick AI assistant access
- Enhanced terminal workflow with AI-powered coding assistance

## Features

### Terminal
- **BossTerm AI Assistant with Welcome Wizard** (#456)
  - Integrated BossTerm AI Assistant directly into the terminal
  - Added Welcome Wizard to guide users through AI assistant setup
  - Improved terminal productivity with AI-powered help (01310bc5)

### AI Assistant
- **Terminal context menu for AI coding assistants** (#454)
  - Added right-click context menu in terminal for quick AI assistant access
  - Provides easy access to popular AI coding tools
  - Streamlined AI integration workflow (ef940487)
