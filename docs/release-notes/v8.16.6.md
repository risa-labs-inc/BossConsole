# BOSS v8.16.6 Release Notes

**Release Date:** 2026-01-24

## Summary

This release delivers critical UI polish improvements, focusing on panel scrollbars, keyboard input handling, and bottom bar refinements for a cleaner, more functional interface.

## Highlights

- Added scrollbars to Console, Git Log, and Git Changes panels for better content navigation
- Fixed keyboard input issue where special characters couldn't be typed in terminal
- Streamlined bottom bar with dynamic terminal title display
- Removed UTF-8 indicator for cleaner status bar
- Improved panel top bar and browser URL bar compactness

## Bug Fixes

### UI
- **Add scrollbars to Console, Git Log, and Git Changes panels** (#534)
  - Console panel now scrollable for better log viewing in constrained spaces
  - Git Log and Git Changes panels gain scrollbars for repository history navigation
  - Prevents content overflow and improves usability in smaller panel sizes

- **Remove UTF-8 indicator from bottom bar** (#523) (#532)
  - Cleaner status bar without UTF-8 encoding display
  - Reduces visual clutter in bottom bar interface

- **Display dynamic terminal title in bottom bar** (#530)
  - Replaces hardcoded "Terminal" text with actual terminal title
  - Shows current directory or running command context
  - Improves user awareness of active terminal state

- **Make sidebar panel top bar and browser URL bar more compact** (#521) (#529)
  - Reduced vertical spacing in panel headers
  - More screen real estate for content viewing
  - Cleaner, tighter UI layout throughout application

### Keyboard
- **Allow typable characters like '?' to pass through to terminal** (#533)
  - Fixed issue where certain special characters were blocked from terminal input
  - Keyboard shortcuts now properly differentiate between shortcuts and typable text
  - Improves terminal usability for commands requiring special characters

### Git
- **Make git UI state window-specific** (#528)
  - Git panel state now isolated per window to prevent cross-window interference
  - Each window maintains independent Git Log and Git Changes state
  - Resolves state conflicts in multi-window scenarios
