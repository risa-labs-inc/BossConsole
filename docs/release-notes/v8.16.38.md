# BOSS v8.16.38

**Release Date:** 2026-02-12

Plugin system stability improvements and significant distribution size reduction with ~86MB saved by removing bundled JxBrowser platform binary.

## Highlights

- **Smaller Downloads**: Distribution size reduced by ~86MB from removing bundled JxBrowser platform binary
- **Plugin Reflection Fix**: Resolved browser plugin compatibility issue with non-suspend link click emission

## Bug Fixes

- **browser**: Add non-suspend tryEmitLinkClick for plugin reflection (#623)
  - Added `tryEmitLinkClick()` method to fix plugin reflection compatibility
  - Resolved issue where browser plugins couldn't use `emitLinkClick` due to suspend function limitations
  - Enables reliable link click event emission from plugins without coroutine context

## Improvements

- **build**: Remove bundled JxBrowser platform binary (~86MB saved) (#622)
  - Removed bundled platform-specific JxBrowser binaries from distribution
  - JxBrowser now downloads required platform binaries automatically on first launch
  - Significantly reduces download size across all platforms
  - Maintains full browser functionality with on-demand resource fetching
