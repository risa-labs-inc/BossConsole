# BOSS v8.16.11 Release Notes

**Release Date:** 2026-01-25

## Summary

This release adds browser tab fullscreen mode with improved video rendering, allowing users to maximize their browser viewing experience without UI distractions.

## Highlights

- Browser tab fullscreen mode for immersive viewing
- Fixed video rendering issues in fullscreen
- Distraction-free browser experience with hidden UI panels

## Features

### Browser
- **Add tab fullscreen mode with video rendering fix** (#263) (#558)
  - Enter fullscreen mode for individual browser tabs
  - Automatic hiding of UI panels in fullscreen mode
  - Fixed video playback rendering issues when in fullscreen
  - Seamless transition between normal and fullscreen views
