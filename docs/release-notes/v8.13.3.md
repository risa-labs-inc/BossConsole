# BOSS v8.13.3

**Release Date**: November 2025

## Summary

This release introduces version selection and downgrade support, enabling users to switch between BOSS versions. It also includes significant Top of Mind plugin improvements with nested sections, favicons, and collapsible sections, plus enhanced tab bar usability.

## Highlights

- Version selection and downgrade support for managing multiple BOSS versions (#210)
- Top of Mind plugin enhancements with nested sections fix, favicons, and collapsible sections (#203)
- Dynamic plus button positioning in tab bar based on scroll state (#199)
- Visual improvements with vertical dividers between tabs (#198)

## Features

- **Versioning**: Added version selection and downgrade support (Issue #191) (#210)
  - Users can now switch between different BOSS versions
  - Support for downgrading to previous versions
  - Improved version management capabilities

- **Top of Mind Plugin**: Enhanced with multiple improvements (#203)
  - Fixed nested sections rendering
  - Added favicon support for better visual identification
  - Implemented collapsible sections for better organization

- **Tab Bar**: Dynamic plus button positioning (Issue #194) (#199)
  - Plus button position now adapts based on scroll state
  - Improved usability when working with many tabs

- **Tab Bar**: Added vertical dividers between tabs (Issue #193) (#198)
  - Enhanced visual separation between tabs
  - Improved tab bar readability

## Dependencies

- Updated GitHub Actions dependencies:
  - Bumped `gradle/actions` from 4 to 5 (#175)
  - Bumped `actions/download-artifact` from 5 to 6 (#176)
  - Bumped `actions/setup-java` from 4 to 5 (#177)
