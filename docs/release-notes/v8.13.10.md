# BOSS v8.13.10

**Release Date**: November 2025

## Summary

This release improves CI/CD reliability by fixing duplicate tag creation issues in the GitHub release workflow.

## Highlights

- Fixed "tag already exists" failures in release workflow
- Streamlined release process by removing redundant tag creation step

## Bug Fixes

- **Workflow**: Remove redundant create-tag job
  - Fixed "fatal: tag already exists" error (exit code 128) in release workflow
  - Let softprops/action-gh-release handle tag creation automatically via tag_name parameter
  - Eliminated duplicate tag creation attempts between release action and separate create-tag job
  - Simplified workflow with atomic tag creation during release publishing
