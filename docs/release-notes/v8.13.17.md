# BOSS v8.13.17

**Release Date**: December 2024

## Summary

This release addresses build system stability issues and browser URL bar usability improvements, including proper Enter key behavior and a new stop button for navigation control.

## Highlights

- Fixed Gradle task configuration errors affecting build reliability
- Resolved URL bar Enter key behavior to properly commit typed text
- Added browser stop button for better navigation control
- Fixed macOS build dependency issues with CLI script generation

## Bug Fixes

- **Build**: Resolved Gradle task `project.layout` access error in build tasks
  - Fixed configuration phase access to task output causing build failures
  - Improved build system stability and reliability

- **Build**: Added task dependencies for CLI script generation in macOS builds (#239)
  - Fixed missing dependencies causing intermittent macOS build failures
  - Ensured proper task ordering for CLI script creation

- **Browser**: Resolved URL bar reversion and added stop button (#226) (#233)
  - Fixed URL bar Enter key to properly reset editing state
  - Added stop button to halt ongoing page navigation
  - Improved URL editing experience and navigation control
  - Resolved Gradle configuration cache compatibility issues
