# BOSS v8.16.25

**Release Date:** 2026-02-06

Stable release promoting the plugin install wizard with enhanced Help menu access and threading improvements.

## Highlights

- **Help Menu Access**: Plugin Setup Wizard can now be launched manually from Help menu
- **Threading Improvements**: UserDataStorage functions properly use IO dispatcher for file operations
- **Startup Behavior Fix**: Wizard now correctly shows only on first login, not every startup
- **Essential Plugins**: Pre-configured defaults include Terminal, Console, ChatGPT, My Secrets, and Downloads

## Features

- **wizard**: Enhance plugin install wizard with Help menu access and threading fixes (#606)
  - Add "Plugin Setup Wizard..." menu item in Help menu for manual access
  - Fix wizard showing every startup by implementing pending file mechanism for completion tracking before user_data.json exists
  - Refactor UserDataStorage functions to suspend functions with Dispatchers.IO for proper threading
  - Add OtherPlugins step to display all uncategorized plugins
  - Configure essential plugins: Terminal, Console, ChatGPT, My Secrets, Downloads
  - Fix plugin ID mappings to match repository format
  - Update fallback plugin list with correct categories and defaults
