# BOSS v8.17.24

**Release Date:** 2026-03-12

This release is a targeted build fix for Windows ARM64, resolving a compilation failure caused by OOP plugin files being incorrectly included on that platform.

## Highlights

- **Windows ARM64 build restored**: OOP plugin source files are now excluded from compilation on Windows ARM64 where they are unsupported, unblocking ARM64 Windows users from building and running BOSS

## Bug Fixes

- **build**: Fix Windows ARM64 compilation failure by excluding OOP plugin files from the build on that architecture
