# BOSS v8.15.11

**Release Date**: January 11, 2026

## Summary

This release adds browser zoom controls, Git integration in the top bar, enhanced URL bar autocomplete, and Windows ARM64 support for BOSS-branded Chromium.

## Highlights

- Browser tabs now have zoom controls (zoom in, zoom out, reset)
- Git integration added to top bar for repository status and operations
- Right arrow key accepts URL bar autocomplete suggestions
- Windows ARM64 support added to Chromium branding workflow

## Features

### Browser
- **Add zoom controls for browser tabs** (#475)
  - Added zoom in, zoom out, and reset zoom controls to browser tabs
  - Improves accessibility and readability for users
  - Standard keyboard shortcuts supported

- **Add right arrow key to accept URL bar autocomplete** (#474)
  - Right arrow key now accepts URL bar autocomplete suggestions
  - Provides familiar browser navigation experience
  - Complements existing Enter key functionality

### Git Integration
- **Add Git integration to top bar** (#90, #457)
  - New Git panel in top bar shows repository status
  - Quick access to common Git operations
  - Real-time branch and status indicators

### Chromium
- **Add Windows ARM64 support to branding workflow**
  - Extended BOSS-branded Chromium builds to Windows ARM64 platform
  - Ensures brand consistency across all supported Windows architectures
  - Automated branding tool integration for ARM64 binaries

## Bug Fixes

### CI/CD
- **Install Windows SDK on ARM64 runner for branding**
  - Fixed branding tool build failures on Windows ARM64
  - Ensures proper SDK dependencies for native branding operations

- **Write params.json without UTF-8 BOM on Windows ARM64**
  - Fixed JSON parsing errors caused by UTF-8 BOM in configuration files
  - Improves branding workflow reliability on Windows ARM64

- **Use Microsoft OpenJDK for Windows ARM64 (has Java 17)**
  - Fixed Java version availability issues on ARM64 runners
  - Switched to Microsoft OpenJDK distribution for better ARM64 support

- **Use Java 21 for Windows ARM64 (17 not available)**
  - Updated to Java 21 to ensure compatibility with ARM64 build tools
  - Resolved availability constraints for Windows ARM64 builds

- **Remove redundant branding tool rebuild on Windows ARM64**
  - Eliminated unnecessary rebuild steps to speed up CI pipeline
  - Improved build efficiency for Windows ARM64 platform

- **Download branding tool via HTTP instead of git clone**
  - Streamlined branding tool acquisition process
  - Reduced build time and improved reliability

- **Use Windows PowerShell for ARM64 jobs**
  - Fixed shell compatibility issues on Windows ARM64 runners
  - Ensures proper script execution on ARM64 platform

- **Use bash shell for Windows ARM64 jobs**
  - Reverted to bash for better compatibility with existing scripts
  - Standardized shell environment across all platforms
