# BOSS v8.12.20

**Release Date**: December 2024

## Summary

This release fixes a critical version downgrade bug that affected users updating from v8.12.17 to v8.12.18, ensuring version numbers in release artifacts always match the intended version.

## Highlights

- Fixed version downgrade issue where stale VersionConstants.kt caused mismatched version numbers in releases
- Added CI/CD verification to prevent future version mismatches
- Implemented runtime version verification at app startup
- Added downgrade protection to the update installer

## Bug Fixes

### Workflow
- **Resolve Issue #111 - Version downgrade during update** (#113)
  - Fixed root cause where VersionConstants.kt was not regenerated before builds
  - Added `outputs.upToDateWhen { false }` to force regeneration on every build
  - Prevents Gradle from using cached generated files with stale versions
  - Ensures version.properties always syncs to VersionConstants.kt

### CI/CD
- **Added version verification steps** (#113)
  - Verifies version after each platform build (macOS, Linux, Windows)
  - Extracts version from generated VersionConstants.kt
  - Compares against expected version from workflow
  - Fails build with clear error message if mismatch detected

### Updates
- **Downgrade protection** (#113)
  - New `extractVersionFromFilename()` function in UpdateInstaller.kt
  - New `verifyNoDowngrade()` check before installation
  - Prevents installing older versions
  - Allows same-version reinstall for repairs
  - Clear error messages for users

## Improvements

### Workflow
- **Runtime version verification** (#113)
  - New VersionVerifier.kt utility class
  - Detects version mismatches at startup
  - Compares VersionConstants.kt against version.properties
  - Logs detailed warnings if mismatch found
  - Non-blocking diagnostics integrated into CoreAuthService.initialize()

### Documentation
- **Release rebuild guide** (#113)
  - Added docs/RELEASE_REBUILD_GUIDE.md
  - Step-by-step guide for deleting and rebuilding bad releases
  - Troubleshooting guide and prevention measures
