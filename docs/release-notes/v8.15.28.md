# Release Notes - v8.15.28

**Release Date**: January 23, 2026

This release fixes a split view cursor interference issue that affected JxBrowser interactions.

## Highlights

- Fixed split view resize cursor interfering with JxBrowser functionality

## Bug Fixes

- **cursor**: Prevent split view resize cursor from interfering with JxBrowser (#510)
  - Resolved issue where split view resize cursor prevented proper mouse interactions in browser panels
  - Improved cursor handling to avoid conflicts between split pane resizing and browser content interaction
