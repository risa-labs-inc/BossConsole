# BOSS v8.15.9

**Release Date**: January 9, 2026

## Summary

This release focuses on tab management improvements, fixing critical issues with tab drag-and-drop reordering and adding auto-scroll functionality for better UX during tab reorganization.

## Highlights

- Fixed tab drag reorder position calculation for scrolled tab bars
- Added auto-scroll when dragging tabs to the edge of the tab bar
- Fixed drag ghost getting stuck after interrupted gestures
- Fixed macOS Chromium executable permissions issue
- Improved CI for ARM64 Linux builds

## Bug Fixes

### Tabs
- **Correct drag reorder position for scrolled tabs** (#463)
  - Fixed drop indicator appearing at wrong position when dragging tabs in a scrolled LazyRow
  - Added `TabBoundInfo` data class to track actual tab indices
  - Updated `calculateReorderIndex` to return actual index instead of iteration index

- **Prevent drag ghost from getting stuck**
  - Ensures proper cleanup when drag gestures are interrupted
  - Added window focus loss handler to cancel active drags (Alt+Tab, click outside)
  - Updated drag end callback signature for better error handling

- **Address race conditions and performance issues in drag system**
  - Fixed race condition by using per-panel edge scroll callbacks
  - Added 150ms throttling for edge scroll to prevent simultaneous animations
  - Added validation check for negative tab indices

### Chromium
- **Set execute permissions on macOS app bundle** (#464)
  - Fixed "Permission denied" errors for BOSS-branded Chromium on macOS
  - Improved permission detection for `.app/Contents/MacOS/` paths
  - Added permission verification in `isChromiumInstalled()` to detect broken cache

### CI
- **Install unzip on ARM64 runner**
  - Fixed Chromium extraction on Linux ARM64 GitHub runners
  - Added unzip to apt-get install in packaging dependencies step

## Improvements

### Tabs
- **Auto-scroll tab bar when dragging to edge**
  - Tab bar now automatically scrolls when drag ghost reaches edge (60px threshold)
  - Implemented with `ScrollDirection` enum and edge scroll callbacks
  - Throttled at ~60fps for smooth performance
  - Each panel registers its own scroll callback to avoid conflicts
