# BOSS v8.17.1

**Release Date:** 2026-02-15

Stability-focused release reverting WebAuthn entitlements and JxBrowser 8.17.0 upgrade to restore stable state while features undergo additional testing.

## Highlights

- **Stability Restoration**: Reverted to stable codebase from v8.16.44 to ensure reliability while WebAuthn and JxBrowser 8.17.0 undergo further testing

## Bug Fixes

- **core**: Revert WebAuthn entitlements and JxBrowser 8.17.0 upgrade (#630)
  - Reverts commits after v8.16.44 to restore stable state
  - Removed WebAuthn/passkey support in embedded Chromium (temporarily)
  - Rolled back JxBrowser to version 8.16.1
  - Reverted entitlements changes for macOS
  - Ensures stability while features are tested further
