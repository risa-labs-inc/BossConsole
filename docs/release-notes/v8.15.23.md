# BOSS v8.15.23

**Release Date**: January 21, 2026

## Summary

This release fixes browser scroll lag, improves focus mode responsiveness, resolves file tree loading issues, and enhances Linux update reliability.

## Highlights

- Fixed browser scroll lag by removing MoveMouseWheelCallback handler (#500)
- Improved focus mode with reduced default reveal sensitivity from 60px to 30px (#495)
- Fixed stuck loading indicator during codebase indexing in file tree (#497)
- Enhanced Linux update script with BossTerm fallback mechanisms (#496)

## Bug Fixes

### Browser
- **Fixed scroll lag** (#500)
  - Removed MoveMouseWheelCallback handler that was causing scroll performance issues
  - Improves browser responsiveness and smooth scrolling experience

### File Tree
- **Fixed stuck loading indicator** (#497)
  - Resolved issue where loading indicator remained visible during folder indexing
  - Extracted shared utilities for better code organization
  - Ensures proper UI feedback during codebase loading operations

### Updater
- **Linux update script improvements** (#496)
  - Ported BossTerm fallback mechanisms to Linux update script
  - Enhances reliability of update installations on Linux platforms
  - Ensures consistent update behavior across all platforms

## Improvements

### Focus Mode
- **Reduced reveal sensitivity** (#495)
  - Changed default reveal sensitivity from 60px to 30px
  - Provides more responsive title bar reveal in focus mode
  - Better balance between distraction-free work and UI access
