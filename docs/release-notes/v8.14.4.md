# BOSS v8.14.4

**Release Date**: December 2025

## Summary

This release introduces BossEditor, a completely custom-built code editor with advanced features including LSP integration, semantic highlighting, code completion, multi-caret editing, and intelligent code folding. BossEditor replaces RSyntaxTextArea as the default editor, offering superior performance and a modern editing experience.

## Highlights

- **BossEditor enabled by default** - Complete custom code editor implementation with 19 development phases
- **LSP and PSI integration** - Full language server protocol support with Kotlin PSI-based semantic analysis
- **Advanced editing features** - Code completion, inlay hints, multi-caret editing, and intelligent code folding
- **Rich visual feedback** - Minimap, rainbow brackets, error squiggles, hyperlinks, and gutter icons
- **40+ syntax highlighting lexers** - Comprehensive language support with semantic highlighting

## Features

- **Enable BossEditor by default** (#381)
  - Custom code editor is now the default editing experience

- **Add LSP integration with code completion and inlay hints** (Phase 18)
