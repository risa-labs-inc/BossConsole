# BOSS v8.13.7

**Release Date**: November 2025

## Summary

This release fixes a critical macOS notarization issue that prevented the app from being properly signed and distributed.

## Highlights

- Fixed macOS code signing failure after CLI script installation (#217)

## Bug Fixes

- **macOS**: Re-sign app bundle after CLI script installation to prevent notarization failure (#217)
  - Added re-signing logic to extractCLIToAppResources task
  - Signs entire app bundle with --force --deep after adding CLI script
  - Verifies signature integrity with codesign verification
  - Respects DISABLE_MACOS_SIGNING environment variable
  - Fixes Apple notarization error 4000: 'Archive contains critical validation errors'
  - Resolves 'The signature of the binary is invalid' rejection by ensuring app bundle seal remains intact after file modifications
