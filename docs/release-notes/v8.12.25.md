# BOSS v8.12.25 Release Notes

Released: December 2025

## Summary

This release introduces a plugin lifecycle system with reset functionality and fixes a critical tab cloning issue, along with updates to the Kotlin runtime.

- New plugin lifecycle system with Reset functionality for improved plugin management (Issue #115)
- Fixed new tab cloning issue that caused improper state duplication (Issue #123)
- Updated Kotlin to version 2.2.21

## Bug Fixes

### Tabs

## Dependencies
