# BOSS v8.13.4

**Release Date**: November 2025

## Summary

This release introduces a dedicated keyboard shortcuts management tab in Settings, providing users with a comprehensive view and control over all application shortcuts in one centralized location.

## Highlights

- New keyboard shortcuts management tab in Settings (#209)
- Comprehensive view of all available shortcuts
- Improved discoverability and customization of keyboard commands

## Features

- **Settings**: Added comprehensive keyboard shortcuts tab (#209)
  - Centralized location for viewing all keyboard shortcuts
  - Organized shortcuts by category for easy navigation
  - Enhanced discoverability of available keyboard commands
  - Improved user control over shortcut management
