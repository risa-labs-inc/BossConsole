# BOSS v8.13.31

**Release Date**: December 2025

## Summary

This release improves terminal link handling to use BOSS browser instead of system browser, fixes Linux browser session persistence, and enhances Linux package installation and updates.

## Highlights

- Terminal links now open in BOSS browser instead of system browser
- Fixed browser session persistence issue on Linux
- Improved Linux DEB/RPM update installation support
- Comprehensive threading documentation rewrite

## Features

### Terminal
- **Open terminal links in BOSS browser instead of system browser** (#337, #333)
  - Terminal links now open in the integrated BOSS browser for a seamless experience
  - Added robust error handling for terminal link clicks
  - Extracted helper functions for better code organization and maintainability

## Bug Fixes

### Browser
- **Browser session not persisting on Linux** (#336)
  - Fixed issue where browser sessions were not persisting correctly on Linux
  - Ensures users maintain their browsing state across application restarts

### Linux
- **Add Linux DEB/RPM update installation support** (#334, #332)
  - Added support for installing updates directly from DEB and RPM packages
  - Improved update script robustness to handle edge cases and failures
  - Set window icon and WM_CLASS for proper local development builds
  - Fixed StartupWMClass to use correct case (BOSS not boss)
  - Install icon to hicolor theme for proper dock integration
  - Fixed Linux installation commands to auto-fetch latest release

## Documentation

- **Comprehensive rewrite of THREADING.md**
  - Complete overhaul of threading documentation for better clarity
  - Added best practices and common pitfall guidance
  - Improved examples and explanations for threading patterns
