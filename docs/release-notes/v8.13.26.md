# BOSS v8.13.26

**Release Date**: December 2025

## Summary

This major release migrates the terminal implementation to the BossTerm library, adds ARM64 Linux support alongside AMD64, and introduces significant build system improvements including native binary signing for macOS notarization.

## Highlights

- Migrated terminal to BossTerm library for improved performance and maintainability (#301)
- Added ARM64 Linux support alongside AMD64 builds
- Enhanced build system with PTY4J native binary signing for macOS notarization
- Start main window maximized by default for better user experience

## Features

### Terminal
- **Migrate terminal to BossTerm library** (#301)
  - Complete migration from custom terminal implementation to BossTerm library
  - Improved terminal performance and reliability
  - Better maintainability through library consolidation

### Platform Support
- **Add ARM64 Linux support alongside AMD64**
  - BOSS now supports both ARM64 and AMD64 architectures on Linux
  - Provides native builds for ARM-based Linux systems
  - Maintains backward compatibility with existing AMD64 installations

### UI
- **Start main window maximized by default**
  - Improved initial user experience with full-screen workspace
  - Better space utilization on first launch

## Bug Fixes

### Build System
- **Sign PTY4J native binaries for macOS notarization + AMD64 Linux naming**
  - Fixed macOS notarization issues by signing PTY4J native binaries
  - Corrected AMD64 Linux package naming conventions
  - Ensures smooth installation on all supported platforms

- **Fail build if PTY4J native binary signing fails**
  - Added build validation to catch signing failures early
  - Prevents distribution of unsigned binaries that would fail notarization

- **Allow release to proceed when ARM64 build is cancelled**
  - Improved CI/CD resilience for multi-architecture builds
  - Prevents entire release from failing if ARM64 build is cancelled

- **Update auto-updater to handle new architecture-specific file names**
  - Auto-updater now correctly detects and downloads architecture-specific builds
  - Ensures seamless updates across different architectures

- **Revert Windows MSI naming for backwards compatibility**
  - Restored original Windows MSI naming convention
  - Maintains compatibility with existing update mechanisms

### UI
- **Synchronize focus mode animations with panel resize**
  - Fixed animation timing issues in focus mode
  - Smoother visual transitions when panels resize

## Dependencies

- **Combined dependency updates** (#297)
  - Updated multiple dependencies for improved performance and security
  - Includes latest stable versions of core libraries

- **Bump ch.qos.logback:logback-classic to 1.5.22** (#300)
  - Updated logging framework to latest version
  - Includes bug fixes and performance improvements
