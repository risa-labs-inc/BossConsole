# BOSS v8.13.29

**Release Date**: December 2025

## Summary

This release fixes critical ARM64 Linux packaging dependencies and updates the integrated terminal with the latest BossTerm improvements.

## Highlights

- Fixed missing packaging dependencies for ARM64 Linux builds
- Updated BossTerm to version 1.0.48

## Bug Fixes

### Linux
- **Add missing packaging dependencies for ARM64 Linux builds** (#316)
  - Fixed ARM64 Linux package builds that were failing due to missing dependencies
  - Ensures proper installation and functionality on ARM64 Linux systems

## Dependencies

- **Update BossTerm to 1.0.48** (#315)
  - Updated integrated terminal component to latest version
