# BOSS v8.12.0

**Release Date**: October 2024

## Summary

This major release introduces simplified WebAuthn authentication, cross-device passkey support, enhanced platform compatibility with Apple Silicon support, and comprehensive dependency updates including Kotlin 2.2.10 and Ktor 3.3.0.

## Highlights

- Simplified WebAuthn flows following W3C standards with improved cross-device authentication
- Apple Silicon hardened runtime support with PTY4J 0.13.9 upgrade
- Major dependency updates: Kotlin 2.2.10, Ktor 3.3.0, AndroidX Lifecycle 2.9.3
- Enhanced CI/CD automation with Claude Code GitHub workflow

## Features

### Authentication

**WebAuthn Simplification** ([#29](https://github.com/risa-labs-inc/BossConsole/pull/29))
- Refactored WebAuthn flows to follow W3C standards
- Improved cross-device authentication reliability
- Enhanced passkey registration and verification processes
- Better error handling and user feedback

**CI/CD Automation** ([#31](https://github.com/risa-labs-inc/BossConsole/pull/31))
- Added Claude Code GitHub workflow for automated development tasks
- Streamlined development and release processes

## Bug Fixes

### Platform Compatibility

**Apple Silicon Support** ([#22](https://github.com/risa-labs-inc/BossConsole/pull/22))
- Upgraded PTY4J to 0.13.9 for Apple Silicon hardened runtime compatibility
- Resolved native library compatibility issues on ARM64 macOS
- Improved terminal functionality on Apple Silicon Macs

**Auto-Update Process**
- Removed unnecessary backup logic from auto-update process
- Fixed bundle versioning for proper version tracking
- Corrected version comparison logic in update service

## Improvements

### Repository Maintenance

**Cleanup**
- Cleaned up gitignore configuration
- Removed redundant homebrew-cask-fork directory
- Improved repository structure and organization
