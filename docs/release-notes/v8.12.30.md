# BOSS v8.12.30 Release Notes

Released: December 2025

## Summary

This release brings comprehensive Console panel improvements with enhanced log capture, better styling, and text selection capabilities, plus Terminal enhancements with emoji support and multi-line text selection.

## Highlights

- Console panel redesigned with Fluck-style UI and complete log history from app startup (#144)
- Multi-line text selection enabled in both Console and Terminal panels (#144)
- Fixed emoji rendering in Console logs with proper UTF-8 decoding (#144)
- Added emoji support to Terminal default font configuration (#144)

## Features

### Console Panel
- **Complete Log History**: Implemented GlobalLogCapture singleton to capture logs from app startup, showing complete history when panel opens (#144)
- **Redesigned UI**: Changed icon to outlined Info, redesigned filter dropdown and search box to match Fluck URL bar styling (28dp height) (#144)
- **Emoji Support**: Removed monospace font to enable emoji rendering (🎬, 🛑, ✅, etc.) (#144)
- **Text Selection**: Added multi-line text selection across all log entries for easy copying of stack traces and error messages (#144)

### Terminal
- **Text Selection**: Enabled multi-line text selection and copy/paste functionality for terminal output (#144)
- **Emoji Support**: Added Noto Color Emoji font to default font fallback for proper emoji rendering (#144)

## Bug Fixes

### Console Panel
- **Crash Fix**: Fixed LazyColumn duplicate key crash by switching to itemsIndexed with guaranteed unique keys using list index + timestamp pattern (#144)
- **UTF-8 Decoding**: Fixed emoji rendering by properly decoding multi-byte UTF-8 sequences (changed from byte-to-char conversion to ByteArrayOutputStream) (#144)
- **Thread Safety**: Prevented deadlock in log listener notification by implementing copy-on-read pattern and invoking callbacks outside synchronized block (#144)

### Terminal
- **Text Selection**: Fixed text selection by moving clickable modifier to outer Box to prevent mouse event consumption (#144)
- **Cursor Positioning**: Corrected cursor positioning with scrollback content by calculating absolute cursor position including scrollback offset (#144)
