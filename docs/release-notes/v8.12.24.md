# BOSS v8.12.24

**Release Date**: December 2024

## Summary

This release introduces native macOS MenuBar integration to BOSS windows, providing a more native macOS application experience with standard menu bar functionality.

## Highlights

- Added native macOS MenuBar to all BOSS windows for improved platform integration

## Features

### macOS
- **Native macOS MenuBar integration** (#122)
  - Added native macOS menu bar to BOSS application windows
  - Provides standard macOS menu structure and keyboard shortcuts
  - Improves platform-native look and feel on macOS
  - Enhances accessibility and discoverability of application features
