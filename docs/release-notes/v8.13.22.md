# BOSS v8.13.22

**Release Date**: December 2024

## Summary

This release enhances window management with native macOS title bar behavior and improves panel usability with better minimum size constraints.

## Highlights

- Title bar double-click now maximizes/restores windows on macOS
- Full screen menu items now respond to window state changes
- Panels now maintain a 20dp minimum size to prevent unusable collapsed states

## Features

- **Window Management**: Added title bar double-click to maximize/restore windows, matching native macOS behavior (#266)
- **Window Management**: Full screen menu items now properly react to window state changes (#266)

## Bug Fixes

- **Panels**: Added 20dp minimum panel size constraint to prevent panels from collapsing into unusable states (#248) (#260)
