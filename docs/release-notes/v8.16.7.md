# BOSS v8.16.7 Release Notes

**Release Date:** 2026-01-24

## Summary

This release fixes a critical bug where scrollbar visibility settings weren't being applied correctly to all panels.

## Highlights

- Fixed scrollbar settings to properly apply to Console, Git Log, and Git Changes panels

## Bug Fixes

### UI
- **Make scrollbar settings actually apply to all panels** (#536)
  - Fixed regression where scrollbar visibility settings weren't being respected
  - Scrollbar preferences now correctly control visibility in Console, Git Log, and Git Changes panels
  - Ensures consistent UI behavior across all panels based on user preferences
