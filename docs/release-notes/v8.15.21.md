# BOSS v8.15.21

**Release Date**: January 20, 2026

## Summary

This release adds memory-efficient large file viewing capabilities to BossEditor, enhances update reliability with automatic fallback mechanisms, and includes comprehensive dependency updates.

## Highlights

- New LargeFileViewer in BossEditor for memory-efficient viewing of files over 10MB with on-demand chunk loading (#489)
- Automatic fallback to public repository updates when GitHub token expires, ensuring uninterrupted update checks (#490)
- Comprehensive dependency updates including JetBrains Markdown, Ktor, Kotlin Wrappers, Korlibs, and Compose resources

## Features

### BossEditor
- **Memory-efficient large file viewer** (#489)
  - Added LargeFileViewer component for files exceeding 10MB threshold
  - Implements on-demand chunk loading to prevent memory exhaustion
  - Provides read-only viewing with file size display and scrollable interface
  - Automatic fallback from full editor to viewer based on file size

## Bug Fixes

### Updater
- **Automatic public repo fallback for expired tokens** (#490)
  - Added automatic fallback to public repository updates when GitHub token expires
  - Prevents update check failures caused by expired or invalid authentication tokens
  - Ensures continuous update availability without manual intervention

## Dependencies

- Updated org.jetbrains:markdown from 0.7.3 to 0.8.0
- Updated io.ktor:ktor-bom from 3.0.3 to 3.0.4
- Updated org.jetbrains.kotlin-wrappers:kotlin-wrappers-bom from 1.0.0-pre.855 to 1.0.0-pre.857
- Updated com.soywiz:korlibs-bom from 6.0.3 to 6.0.4
- Updated org.jetbrains.compose:compose-resources-generator-gradle-plugin from 1.7.4 to 1.7.5
- Updated org.jetbrains.compose:compose-gradle-plugin from 1.7.4 to 1.7.5
