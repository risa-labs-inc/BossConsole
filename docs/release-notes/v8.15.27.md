# Release Notes - v8.15.27

**Release Date**: January 23, 2026

This release fixes a critical macOS fullscreen issue and significantly improves the README documentation with better onboarding for new users.

## Highlights

- Restored macOS fullscreen functionality with transparent title bar
- Major README overhaul with badges, download links, and quick install commands
- Fixed non-responsive buttons in fullscreen mode on macOS

## Bug Fixes

- **window**: Restore macOS transparent title bar for fullscreen support (#508)
  - Fixed non-responsive top bar buttons in fullscreen mode on macOS
  - Restored macOS-specific window properties: fullscreenable, fullWindowContent, transparentTitleBar, windowTitleVisible
  - Restored BossTitleBar component for custom title bar UI that works with transparent native title bar
  - Clicks no longer intercepted by native macOS UI layer in fullscreen

## Improvements

- **docs**: Update README to align with BossConsole-Releases (#509)
  - Added badges for version, build status, platform, and license
  - Added prominent link to BossConsole-Releases for pre-built downloads
  - Added download table with links to all platform installers
  - Added quick install commands (Homebrew, curl script, PowerShell)
  - Streamlined developer documentation (build, config, version management)
  - Added CI/CD workflows table and release instructions
  - Added CLI usage examples
  - Added support section with links to issues, releases, and enterprise
