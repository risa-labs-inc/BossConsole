# BOSS v8.15.8

**Release Date**: January 7, 2026

## Summary

This release focuses on browser navigation stability, UI polish, and expanded platform support with Linux ARM64. It also resolves 139 build warnings across the codebase for improved code quality.

## Highlights

- Fixed browser navigation getting stuck on rapid back/forward clicks
- Added Linux ARM64 support with BOSS-branded Chromium auto-download
- Improved Downloads window UI with full dark background
- Resolved 139 build warnings across the entire codebase
- Fixed sidebar positioning and Dashboard display in browser tabs

## Features

### Platform Support
- **Linux ARM64 support** (b39d01d9)
  - Added BOSS-branded Chromium auto-download for ARM64 architecture
  - Integrated ARM64 into Chromium branding workflow (cd75a45f)
  - Fixed workflow issues for Linux ARM64 builds (2eaef9be, a80e46bc)

## Bug Fixes

### Browser
- **Prevent navigation getting stuck on rapid clicks** (#462)
  - Fixed navigation state corruption when rapidly clicking back/forward buttons
  - Improved state management to handle quick successive navigation actions (3af9fd11, f6059209)

### UI
- **Download window improvements** (d69e9ef5)
  - Downloads window now fills entire window with proper dark background
  - Enhanced visual consistency with the rest of the application

### Panel
- **Sidebar and Dashboard fixes** (#460)
  - Fixed sidebar positioning issues
  - Resolved Dashboard display problems when opened in browser tabs (ad8d0255)

## Improvements

### Code Quality
- **Resolve 139 build warnings** (#461)
  - Cleaned up warnings across the entire codebase
  - Improved code maintainability and build output clarity (b6afe63d)
