# BOSS v8.16.9 Release Notes

**Release Date:** 2026-01-25

## Summary

This release introduces keyboard shortcuts for split tab operations, Google search integration in context menus, and unified context menu styling with a popup-based New Tab Dialog.

## Highlights

- New keyboard shortcuts for split tab operations (split horizontally, vertically, and join splits)
- "Search with Google" option added to browser context menus
- Unified context menu styling with modern popup-based New Tab Dialog

## Features

### Keyboard Shortcuts
- **Add keyboard shortcuts for split tab operations** (#284) (#552)
  - Split current tab horizontally with keyboard shortcut
  - Split current tab vertically with keyboard shortcut
  - Join splits with keyboard shortcut
  - Improves efficiency for power users managing complex layouts

### Browser
- **Add "Search with Google" context menu option** (#551)
  - Right-click on selected text to search with Google
  - Opens search results in new tab
  - Seamless integration with existing browser context menu

### UI
- **Unified context menu styling and Popup-based NewTabDialog** (#323) (#555)
  - Consistent visual design across all context menus
  - New Tab Dialog now uses modern popup component
  - Improved user experience with cohesive UI patterns
