# Release Notes - v8.16.0

**Release Date**: January 23, 2026

This release brings comprehensive video playback support and chrome:// URL navigation to the integrated browser.

## Highlights

- Enabled proprietary video codecs (H.264, AAC, HEVC) for full video playback support
- Added Widevine DRM support for streaming services (Netflix, Disney+, Spotify)
- Implemented chrome:// URL support for accessing browser internals
- GPU acceleration and hardware video decoding improvements

## Features

- **browser**: Enable proprietary codecs for video playback (#513)
  - Added H.264 and AAC codec support to fix video playback issues on GitHub, YouTube, and other platforms
  - Enabled HEVC (H.265) codec for efficient high-quality video playback
  - Activated Widevine DRM for protected content streaming services

- **browser**: Add chrome:// URL support and GPU acceleration (#513)
  - Added chrome:// scheme recognition enabling navigation to chrome://gpu, chrome://settings, and other internal pages
  - Enabled GPU rasterization and zero-copy for improved rendering performance
  - Added Linux-specific VaapiVideoDecoder/Encoder flags for hardware video decode
  - Enabled overscroll history navigation for touchscreen devices

## Improvements

- **docs**: Remove build status badge from README (#511)
  - Cleaned up README by removing inactive build status badge
