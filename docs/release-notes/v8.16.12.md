# BOSS v8.16.12 Release Notes

**Release Date:** 2026-01-25

## Summary

This release focuses on improving dashboard functionality and code organization with plugin architecture refactoring. Dashboard metrics now properly track file, browser, and terminal activity in real-time.

## Highlights

- Dashboard metrics fully integrated with file operations, browser navigation, and terminal sessions
- Plugin system extracted to 31 separate Gradle modules for better maintainability
- Major dependency updates including Ktor 3.4.0, kotlinx-serialization 1.10.0, and Gradle 9.3.0

## Bug Fixes

### Dashboard
- **Wire up dashboard metrics to track file, browser, and terminal activity** (#554) (#559)
  - Fixed dashboard statistics remaining at zero due to missing integration points
  - Added DashboardStatsManager calls to FileEventBus, FluckTabInfo, and terminal handlers
  - All metrics now update in real-time via StateFlow and persist to ~/.boss/dashboard-stats.json

## Improvements

### Architecture
- **Extract plugins to separate Gradle modules** (#541) (#560)
  - Created 31 plugin modules organized by category: core infrastructure (8), type modules (4), panel modules (19)
  - Each plugin follows consistent pattern: Info Object, Plugin Object, and Component
  - Added dynamic Git panel registration that hides when no project or non-git repository
  - Fixed UI differences in TopOfMind, Performance, SecretManager, Codebase, and Git Log panels
  - Added comprehensive plugin system documentation

## Dependencies

- **Ktor**: 3.3.3 → 3.4.0
- **kotlinx-serialization**: 1.9.0 → 1.10.0
- **Gradle Wrapper**: 9.2.0 → 9.3.0
- **Logback**: 1.5.25 → 1.5.26
