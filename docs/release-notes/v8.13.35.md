# BOSS v8.13.35

**Release Date**: December 2025

## Summary

This release fixes a critical bug in tab context menu actions that prevented menu items from executing properly due to a trailing lambda parameter ordering issue.

## Highlights

- Fixed context menu actions not executing in tab context menus
- Refactored ContextMenuItem parameters for better Kotlin DSL ergonomics

## Bug Fixes

### Tabs
- **Context menu actions not executing due to trailing lambda bug** (#360)
  - Fixed parameter ordering in ContextMenuItem to place onClick as the last parameter
  - Enables proper trailing lambda syntax in Kotlin
  - Resolved issue where context menu items appeared but didn't execute their actions

## Improvements

### Tabs
- **Reorder ContextMenuItem params so onClick is last for trailing lambda**
  - Refactored ContextMenuItem to follow Kotlin convention of placing function parameters last
  - Improves code readability and maintainability
  - Enables cleaner DSL-style usage
