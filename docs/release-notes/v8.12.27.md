# BOSS v8.12.27 Release Notes

Released: December 2025

## Summary

This release fixes a critical browser usability issue where hyperlinks and popup windows opened in detached OS browser windows instead of BOSS tabs, disrupting the integrated browsing experience.

## Highlights

- Fixed hyperlinks with `target="_blank"` opening in detached OS windows (Issue #137, #140)
- Implemented popup interception to open new tabs within BOSS instead of external windows
- Added supabase/.temp to gitignore for cleaner version control

## Bug Fixes

### Browser
- **Popup Window Handling**: Implemented JxBrowser CreatePopupCallback to intercept `target="_blank"` links and `window.open()` calls, ensuring they open as new tabs within BOSS instead of creating detached OS browser windows (Issue #137, #140)

## Improvements

### Development
- **Build Artifacts**: Added supabase/.temp directory to gitignore to exclude temporary Supabase CLI files from version control (#140)
