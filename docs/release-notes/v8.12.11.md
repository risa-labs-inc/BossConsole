# BOSS v8.12.11

**Release Date**: October 2024

## Summary

This release focuses on UI cleanup and bottom bar functionality, removing unimplemented top bar components and fixing critical issues with the bottom bar not updating when tabs are opened or changed.

## Highlights

- Fixed bottom bar to correctly display and update with active tabs
- Removed hardcoded project paths in favor of dynamic project selection
- Cleaned up top bar by hiding unimplemented components
- Created GitHub issues to track future feature implementation

## Bug Fixes

### Bottom Bar
- **Bottom bar now updates correctly when opening files and tabs** (#95)
  - Fixed critical bug where bottom bar always showed "BOSS-Kotlin | Ready" instead of updating with file paths and URLs
  - Changed BossBottomBar to receive active tabsComponent from splitViewState instead of standalone component
  - Bottom bar now shows file breadcrumb when opening files from CodeBase
  - Bottom bar now shows URLs when opening Fluck browser tabs
  - Bottom bar now shows "Terminal" for terminal tabs
  - Bottom bar shows "ProjectName | Ready" when no tabs are active

- **Removed static plugin placeholders from bottom bar**
  - Removed hardcoded project path (`/Users/kshivang/Development/BOSS-Kotlin/`)
  - Now uses `ProjectState.selectedProject` dynamically
  - Replaced static plugin names ("Lanager > Mastery > Task Resolver") with dynamic project context
  - Shows "{ProjectName} | Ready" when no tab is active
  - Shows simple "Ready" text as fallback placeholder

## Improvements

### Top Bar
- **Hide unimplemented top bar components** (#94)
  - Commented out Git branch button and helper functions
  - Commented out Run/Debug controls (BossTopRunBar)
  - Commented out Global search button
  - Created GitHub issues for future implementation: #90 (Git Integration), #91 (Run/Debug Controls), #92 (Global Search), #93 (Lanager Plugin)
  - All TODO comments now reference actual GitHub issue numbers with URLs
  - Documented current top bar components and disabled features in CLAUDE.md
