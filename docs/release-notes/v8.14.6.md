# BOSS v8.14.6

**Release Date**: December 2025

## Summary

This release improves the startup experience by adding an offline screen with retry functionality and updates the terminal integration to the latest version.

## Highlights

- New offline screen with retry functionality on startup
- Graceful handling of network connectivity issues during authentication
- Updated BossTerm to 1.0.66 for improved terminal integration

## Features

### Auth
- **Add offline screen with retry on startup** (#399)
  - Displays user-friendly offline screen when network is unavailable
  - Provides retry functionality to attempt reconnection
  - Improves user experience during connectivity issues
  - Better error handling during authentication flow

## Bug Fixes

### Auth
- **Address code review feedback on offline handling** (#399)
  - Refined offline detection logic
  - Improved error messaging and user feedback
  - Enhanced retry mechanism reliability

## Dependencies

- **Update BossTerm to 1.0.66** (6bb3fe08)
  - Latest terminal integration improvements and fixes
