# BOSS v8.14.8

**Release Date**: December 2025

## Summary

This release delivers major BossEditor polish with Phase 19 features, terminal file hyperlink navigation with line:column support, and critical bug fixes for browser URL recovery and focus mode defaults.

## Highlights

- BossEditor Phase 19 polish features with comprehensive code quality improvements
- Terminal file hyperlinks now support line:column navigation
- Fixed browser tab URL recovery after idle state
- Focus mode now disabled by default for better new user experience

## Features

### Terminal
- **Add line:column navigation for file hyperlinks** (40c355ff)
  - File links now support format `file.txt:10:5` to jump to specific line and column
  - Integrated with BossEditor for precise code navigation
- **Add consistent dialog behavior for file hyperlinks** (87a8e868)
  - Unified dialog experience across terminal file interactions
  - Improved file opening workflow
- **Add file path validation for hyperlinks** (0bca23ea)
  - Prevents errors from invalid or malicious file paths
  - Enhanced security for file link handling

### BossEditor
- **Phase 19 polish features** (#385)
  - Spell checker with custom dictionary support and resource-based word lists
  - Thread-safe dictionary file operations with async I/O
  - Dictionary size limits and proper error handling
  - QuickFix error handling improvements
  - Comprehensive unit tests for spell checking functionality

## Bug Fixes

### Browser
- **Preserve current URL when tab recovers from idle state** (80c9b676, #405)
  - Fixed issue where tabs would lose their URL after recovering from idle
  - Ensures consistent tab state across suspension/recovery cycles
- **Add thread safety for URL navigation state** (7e679fb5)
  - Prevents race conditions in browser navigation
  - Improved stability during rapid tab switching
- **Complete thread safety and add tests for URL recovery** (0452e7cc)
  - Comprehensive thread safety implementation
  - Added unit tests to verify recovery behavior

### Focus Mode
- **Disable focus mode by default for new users** (b9847580, #404)
  - Improved initial user experience
  - Focus mode now opt-in rather than opt-out
- **Add unit tests for FocusModeSettings** (caee0d97)
  - Ensures focus mode settings work correctly
  - Validates default behavior

### BossEditor
- **Address critical code review issues** (f8a5920d, c3419a4c, a5cc555a)
  - Fixed multiple threading and performance concerns
  - Resolved potential deadlocks and resource leaks
- **Add bounds checking and edge case test** (1d5d488f)
  - Prevents array out-of-bounds errors
  - Added comprehensive edge case coverage
- **Remove unreachable dead code in SimpleSpellChecker initialization** (a506ace0)
  - Code cleanup and maintainability improvements
- **Fix potential deadlock in GitService process stream reading** (69ba6315)
  - Resolved threading issue that could cause Git operations to hang
  - Improved reliability of Git integration
- **Make SpellChecker implement Closeable for proper resource cleanup** (2972629b)
  - Ensures spell checker resources are properly released
  - Prevents memory leaks
- **Move hardcoded dictionary to resource file and add async I/O** (e2645164)
  - Better resource management for spell checking dictionaries
  - Non-blocking file operations
- **Add dictionary size limit and QuickFix error handling** (1a632a36)
  - Prevents memory issues from oversized dictionaries
  - Improved error recovery
- **Add thread safety to custom dictionary file operations** (c3419a4c)
  - Safe concurrent access to dictionary files
  - Prevents file corruption

### File Hyperlinks
- **Code quality improvements for file hyperlink handling** (#403)
  - Enhanced reliability and maintainability
  - Better error handling and validation

### Tests
- **Add CI-aware thresholds for performance benchmarks** (3135b3b0, 43e226f0)
  - Adapted performance tests to run reliably in CI environments
  - Prevents false failures due to CI resource constraints

## Improvements

### Code Quality
- **Address code review feedback for Phase 19 features** (1704e4fa)
  - Comprehensive code review improvements
  - Enhanced documentation and test coverage
- **Address code review threading and performance issues** (737f7bdd)
  - Optimized threading patterns
  - Reduced performance overhead

## Dependencies

- **Update BossTerm to 1.0.68** (3032e267)
  - Latest terminal improvements and bug fixes
