# BOSS v8.13.20

**Release Date**: December 2024

## Summary

This release introduces comprehensive download management with auto-closing redirect tabs and resolves all Kotlin compiler deprecation warnings for improved code quality.

## Highlights

- Download Manager with real-time progress tracking and sidebar panel
- Smart auto-close for redirect tabs when downloads start
- Full pause/resume/cancel support for downloads with progress visualization
- Eliminated all Kotlin compiler deprecation warnings with proper API migrations

## Features

- **Browser**: Comprehensive download management system with progress tracking (#257)
  - Downloads sidebar panel with real-time status updates
  - Pause/resume functionality for active downloads (requires server HTTP range support)
  - Cancel button for active downloads, delete button for paused/completed downloads
  - File name sanitization and automatic duplicate handling
  - Smart auto-close for redirect tabs (closes tabs opened within 3 seconds of download start)
  - Orange progress bar for paused downloads, blue for active
  - Memory leak fix: proper cleanup of SpeedCalculator instances when clearing downloads

## Improvements

- **Code Quality**: Migrated from deprecated APIs to modern replacements (#246)
  - Migrated LocalClipboardManager to LocalClipboard API with ClipEntry
  - Migrated deprecated Material Icons (Icons.Outlined) to Icons.AutoMirrored.Outlined
  - Added proper experimental API opt-ins (FlowPreview, DelicateCoroutinesApi)
  - Removed redundant null checks and backwards compatibility code
  - Clean build with zero compiler warnings
