# BOSS v8.16.40

**Release Date:** 2026-02-12

Terminal sidebar plugin stability improvements with bundled distribution and enhanced GitHub URL integration.

## Highlights

- **Plugin Bundling Fix**: Terminal sidebar plugin now properly bundled to prevent stale store version conflicts
- **Enhanced GitHub URLs**: Wizard GitHub URLs now include terminal sidebar reference for better discoverability

## Bug Fixes

- **terminal**: Bundle terminal sidebar plugin to prevent stale store version (#624)
  - Bundled terminal sidebar plugin directly into the distribution
  - Prevents stale store version conflicts that could cause plugin compatibility issues
  - Ensures consistent plugin version across installations

- **terminal**: Add terminal sidebar to wizard GitHub URLs
  - Updated wizard to include terminal sidebar in GitHub repository references
  - Improves discoverability of the terminal sidebar plugin
  - Ensures users can easily access the terminal sidebar source code
