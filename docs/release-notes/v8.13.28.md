# BOSS v8.13.28

**Release Date**: December 2025

## Summary

This release introduces enhanced terminal tab management with state persistence and fixes for terminal rendering, along with dependency updates and improved CI/CD support for ARM64 builds.

## Highlights

- TabbedTerminal with state persistence for main terminal tabs
- Fixed terminal background color and initial command handling
- Updated to Kotlin 2.3.0 and Logback 1.5.23
- Improved Linux ARM64 build infrastructure

## Features

### Terminal
- **Use TabbedTerminal with state persistence for main terminal tabs** (#314)
  - Enhanced terminal tab management with persistent state across sessions
  - Improved reliability and user experience in terminal tab handling

## Bug Fixes

### Terminal
- **Fixed terminal background color and initial command handling**
  - Resolved rendering issues with terminal background color
  - Fixed proper handling of initial commands when opening terminal tabs

## Improvements

### CI/CD
- **Use linux_arm64 larger runner for ARM64 builds**
  - Improved build performance and reliability for Linux ARM64 packages
  - Better resource allocation for ARM64 compilation

## Dependencies

- **Bump Kotlin from 2.3.0-RC2 to 2.3.0** (#313)
  - Updated to stable Kotlin 2.3.0 release
- **Bump ch.qos.logback:logback-classic to 1.5.23**
  - Updated logging framework to latest version
