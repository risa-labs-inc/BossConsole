# BOSS v8.8.0 Release Notes

**Release Date:** 2024-10

## Summary

Initial public release of BOSS (Business Operating System Service) - a desktop application built with Kotlin Multiplatform and Compose Multiplatform featuring integrated browser (Fluck), terminal support, multi-panel layouts, and RPA automation capabilities.

## Highlights

- **JAR Build Support**: Professional distribution packaging with JAR build system
- **LLM-Powered RPA**: Natural language automation with Anthropic Claude integration for browser automation
- **RPA Recorder**: Enhanced UI for recording and executing browser automation workflows
- **Multi-Panel Interface**: Flexible 4-panel layout with split views and dynamic panel management
