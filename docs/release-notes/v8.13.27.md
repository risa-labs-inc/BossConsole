# BOSS v8.13.27

**Release Date**: December 2025

## Summary

This release updates JxBrowser to version 8.15.0 with version consistency fixes and banner styling improvements to match BossTerm's dark theme.

## Highlights

- Updated JxBrowser from 8.13.0 to 8.15.0
- Fixed version inconsistency in build.gradle.kts
- Updated banner styling to match BossTerm's dark theme

## Bug Fixes

### Browser
- **Sync JxBrowser version in build.gradle.kts to 8.15.0** (#304)
  - Fixed version mismatch between JxBrowser dependency declarations
  - Ensures consistent browser engine version across the application

## Improvements

### UI
- **Update banner to match BossTerm's dark theme**
  - Enhanced visual consistency between BOSS and BossTerm
  - Improved dark mode aesthetics

## Dependencies

- **Update JxBrowser from 8.13.0 to 8.15.0** (#304)
  - Upgraded to latest JxBrowser version for improved browser functionality
  - Includes latest Chromium updates and performance improvements
