# BOSS v8.16.14 Release Notes

**Release Date:** 2026-01-27

## Summary

This release modernizes the build configuration by replacing deprecated Compose accessor patterns with direct Maven coordinates, ensuring compatibility with future Compose Multiplatform versions.

## Improvements

### Dependencies
- **Replace deprecated compose.* accessors with direct Maven coordinates** (#566) (76143d3)
  - Migrated from deprecated `compose.desktop.currentOs` to direct Maven coordinate `org.jetbrains.compose.desktop:desktop-jvm`
  - Updated build configuration to use explicit dependency declarations instead of accessor methods
  - Ensures compatibility with upcoming Compose Multiplatform releases
