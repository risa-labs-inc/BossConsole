# BOSS v8.15.19

**Release Date**: January 19, 2026

## Summary

This release resolves a critical macOS ARM64 performance issue and includes comprehensive dependency updates for improved stability.

## Highlights

- Fixed progressive performance degradation on macOS ARM64 by replacing JIT compilation disable with delayed compilation (#484)
- Updated multiple dependencies including Supabase 3.3.0, JNA 5.18.1, Clikt 5.0.3, and Logback 1.5.24

## Bug Fixes

### JVM
- **Replace C2 JIT disable with delayed compilation for macOS ARM64** (#484)
  - Resolved progressive performance degradation (lag) during long sessions
  - Changed from complete C2 JIT disable to delayed compilation (15000 invocations vs default 1500)
  - Only applies JIT tuning on macOS ARM64 where startup crashes occurred
  - Maintains fast startup while enabling full C2 optimization after JxBrowser loads
  - Result: Fast startup + full C2 optimization + no crash

## Dependencies

- Updated Clikt from 4.2.1 to 5.0.3
- Updated actions/upload-artifact from 4 to 5
- Updated actions/checkout from 5 to 6
- Updated Logback from 1.5.23 to 1.5.24
- Updated Supabase from 3.2.6 to 3.3.0 (BOM, PostgREST, Auth, Realtime, Storage, Functions)
- Updated JUnit from 6.0.1 to 6.0.2
- Updated JNA from 5.14.0 to 5.18.1
- Updated BossTerm to 1.0.86
