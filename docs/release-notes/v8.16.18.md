# BOSS v8.16.18 Release Notes

**Release Date**: 2026-01-29

## Summary

CI/CD improvements and security hardening for the release workflow with global search functionality and file operations context menu.

## Highlights

- **Global Search**: Double-shift shortcut activates global search across the application (#92) (#573)
- **Context Menu**: File operations available via right-click in codebase view (#570)
- **Security**: Hardened sync-release.yml workflow against script injection vulnerabilities
- **Workflow Reliability**: Fixed multiple release workflow issues including GitHub Release creation and configuration

## Changes

### Features
- **search**: Add global search functionality with double-shift shortcut (#92) (#573)
- **codebase**: Add context menu with file operations (#570)
- **release**: Add support for unstable version releases (#571)

### Bug Fixes
- **release**: Fix Create GitHub Release step being skipped (5efb09fa)
- **release**: Use multiline mode for PowerShell regex patterns (d872061d)
- **release**: Remove invalid 'secrets: inherit' from workflow_call definition (2f3a0033)
- **release**: Remove run-name to fix workflow failures (b77e551f)
- **release**: Add dynamic run-name to release workflow (bc8e2006)

### Security
- **workflow**: Harden sync-release.yml against script injection (e91fb5de)
