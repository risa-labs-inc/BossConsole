# BOSS v8.15.18

**Release Date**: January 12, 2026

## Summary

This release addresses Windows-specific path handling issues and improves user communication around browser engine downloads.

## Highlights

- Fixed path separator and PowerShell command syntax issues on Windows (#478)
- Improved browser engine download messaging (#479)

## Bug Fixes

### Windows
- **Fix path separator and PowerShell command syntax issues** (#478)
  - Resolved Windows-specific path separator handling
  - Fixed PowerShell command syntax for improved cross-platform compatibility
  - Enhanced reliability of Windows command execution

### UI
- **Update browser engine download copy** (#479)
  - Improved messaging and user feedback during browser engine downloads
  - Enhanced clarity of download process communication
