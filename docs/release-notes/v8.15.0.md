# BOSS v8.15.0

**Release Date**: January 2, 2026

## Summary

This major release introduces BOSS-branded Chromium support and completes the LSP integration for BossEditor, enabling advanced code intelligence features.

## Highlights

- BOSS-branded Chromium builds with custom branding workflow
- Complete LSP integration with structured logging and configuration system
- Updated BossTerm to version 1.0.71
- Dynamic JxBrowser version management for Chromium downloads

## Features

### Browser
- **Add BOSS-branded Chromium support** (#428)
  - Implemented custom Chromium branding workflow (d11270ba, 4363649a)
  - Integrated JxBrowser Gradle plugin for proper repository access (f2f90563)
  - Added dynamic JxBrowser version handling for Chromium downloads (62c50ff7)
  - Support for building branded Chromium from BossConsole (4363649a)

### Editor
- **Complete LSP integration for BossEditor** (#429)
  - Full Language Server Protocol support (efa8dddc)
  - Added structured logging and configuration system (#430) (7f724abe)

### UI
- **Update plugin icons for better visual identity** (01647b99)

## Bug Fixes

### Workspace
- **Use correct Claude CLI flag --resume instead of --continue** (#431) (014e3a15)

### Chromium Build
- Handle Linux headless CI and create executable.name (489237d8)
- Fix Linux sandbox and macOS secret names (7fea87e9)
- Use desktop source set for KMP project (4ccf26b4)
- Use project's downloadChromium task with proper license (e50db92f)
- Use macos-15-large runner for Intel builds (b900b0b5)
- Simplify branding workflow with placeholders (788ef804)

## Dependencies

- **BossTerm updated to 1.0.71** (#418) (430f8565)
