# BOSS v8.13.32

**Release Date**: December 2025

## Summary

This release introduces comprehensive session performance monitoring with a dedicated panel, IntelliJ-style run configurations with history and auto-detection, and improved project management with persistence across sessions.

## Highlights

- New performance monitoring system with real-time CPU, memory, and resource tracking
- IntelliJ-style run configurations with smart detection for Gradle and Cargo projects
- Project persistence with automatic restore and recent projects list
- Enhanced code editor with gutter run buttons and improved visual design

## Features

### Performance Monitoring
- **Session performance monitoring with status bar indicator and panel** (#343, cf8e19fe)
  - Real-time CPU, memory, thread, and resource usage tracking
  - Tabbed interface: Overview, CPU, Memory, Timings, Resources
  - Circular gauges with visual indicators for key metrics
  - Clickable overview items navigate to detailed tabs
  - Export metrics functionality with automatic file opening
  - Performance settings in Settings UI with configurable collection intervals
  - Throttled StateFlow updates (every 10 seconds) for efficiency
  - Platform-aware resource providers with thread-safe registration
- **Detailed resource information** (c3b222a7)
  - Thread details in CPU tab
  - Detailed memory breakdown
  - Timing information for operations
  - Resource utilization tracking
- **Performance settings section** (a063236d)
  - Configurable monitoring intervals
  - Reset to defaults option with styled button

### Run Configurations
- **Run Configurations detection plugin** (#343, 35f0efec)
  - IntelliJ-style run history in dropdown with delete buttons
  - Smart run command generation with Gradle/Cargo module detection
  - Commands now target specific files instead of generic builds
  - Auto-detect project root for run commands
  - No default selection, deduplicate commands, unique names
  - Working directory set to project root
  - Initial command passed to terminal when running configurations
- **Editor gutter run button** (4af7bffe)
  - Colored run/stop icons with configurable size in top bar
  - IntelliJ's run icon color (#59A869) for consistency
  - 24dp icon size for better visibility
  - Synced line numbers with content
  - Zap icon for type icon, PlayArrow for run button

### Project Management
- **Persist recent projects to disk** (#341, 5ffb1b8c)
  - Save and restore project path with workspaces
  - Recent projects list maintained across sessions
  - 'Open Project' shown in top bar when no project selected
  - Thread-safe implementation with proper error handling

### Browser
- **CodeBase plugin IntelliJ patterns and JxBrowser session persistence fix** (#341, 40e7aa87)
  - Improved codebase search patterns
  - Fixed browser session restoration issues

### Console
- **Info button in status bar now opens Console panel** (c301e18b)
  - Quick access to console output from status bar

## Bug Fixes

### Run Configurations
- **Address security and reliability issues** (#343, ad6a10a7)
  - Improved command validation
  - Better error handling for invalid configurations
- **Improve run configuration UX** (b91f4a73)
  - No default selection to prevent accidental runs
  - Deduplicate commands to avoid confusion
  - Ensure unique names for better organization

### Performance Monitoring
- **Address memory leak and exception handling** (#343, 5525a12f)
  - Fixed memory leaks in PerformanceMonitor
  - Comprehensive exception handling
  - Optimize StateFlow updates to reduce stale data
- **Fix file path disclosure** (1f73b257)
  - Sanitized file paths in performance monitoring output
- **Address thread safety issues** (#341, ae046c7a)
  - Fixed race conditions in history buffer
  - Proper synchronization for concurrent access
  - Windows path handling improvements
- **History buffer size and StateFlow update optimization** (#339, 9718ddf0)
  - Optimized buffer management
  - Reduced unnecessary updates

### Updates
- **Handle Linux-deb and Linux-rpm in getExpectedAssetName()** (#338, d829d1ea)
  - Fixed update detection for Linux DEB and RPM packages
  - Added platform info to update asset debug log

### Input Handling
- **Platform-aware keyboard modifier mapping for Linux/Windows** (d69d5888)
  - Fixed keyboard shortcuts on Linux and Windows
  - Proper modifier key detection per platform

## Improvements

### Performance
- **Cache getAllPanels() to avoid repeated tree traversals** (#343, 6e6bd48b)
  - Significant performance improvement for panel operations
- **Use LazyColumn items() instead of forEach for list rendering** (#343, 06157597)
  - More efficient list rendering with better memory usage
- **Throttle history StateFlow updates to every 10 seconds** (#339, 16ac1e40)
  - Reduced CPU usage from excessive state updates

### Code Quality
- **Replace printStackTrace with println** (#343, 6149b433)
  - Cleaner error logging throughout codebase
  - Document memory usage patterns
- **Clean up verbose form-related debug logs** (baa407d0)
  - Reduced log noise for better debugging experience

### UI/UX
- **Make performance panel more compact** (571e2393)
  - Compact tabs with increased element sizes
  - Improved visual hierarchy
- **Visual enhancements to performance panel** (39b299ef)
  - Better color coding and visual feedback
  - Larger circular gauges (150dp) for readability
