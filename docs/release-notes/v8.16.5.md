# BOSS v8.16.5 Release Notes

**Release Date:** 2026-01-24

## Summary

This release fixes a critical keyboard shortcut reliability issue by establishing MenuBar as the single source of truth for all shortcuts, eliminating duplicate handlers and improving cross-platform consistency.

## Highlights

- Fixed keyboard shortcuts not working in browser and terminal contexts
- Eliminated duplicate shortcut handlers that caused multi-window conflicts
- Added status message display for workspace save confirmation
- Improved panel slot-to-content mapping for Run Configurations

## Bug Fixes

### Keyboard Shortcuts
- **Use MenuBar as single source of truth for shortcuts** ([#527](https://github.com/risa-labs-inc/BossConsole/pull/527))
  - Fixed browser shortcuts (Cmd+R/N/T/W, Cmd+Shift+F/S) not working when JxBrowser had focus
  - Added AWT-level keyboard interceptor to enable shortcuts in terminal tabs
  - Fixed Cmd+W to properly close window when no tabs are open
  - Cross-platform modifier detection (Cmd on macOS, Ctrl on Windows/Linux)
  - Removed duplicate "Close Tab" entry from Window menu
  - Fixed race condition in KeyboardEventBus test with proper cleanup synchronization

### Panel Management
- **Correct slot-to-panel mapping for bottom panels**
  - Fixed incorrect mapping where left.bottom and right.bottom slots were mapping to top panels
  - Resolves Run Configurations panel opening in wrong location when project is selected
  - Prevent panel registry updates from overwriting panel content during workspace restore

### Memory Management
- **Window state cleanup to prevent memory leaks**
  - Added cleanupWindow() method to remove entries from state maps when windows close
  - Proper lifecycle management with DisposableEffect hooks

## Improvements

### UI/UX
- **Status message display in bottom bar**
  - Shows temporary status messages (e.g., "Workspace Saved") with 2-second auto-dismiss
  - Thread-safe message handling with proper coroutine lifecycle
- **Keyboard Shortcuts added to Help menu**
  - New menu item with ? (Shift+/) shortcut to show keyboard shortcuts dialog

### Architecture
- Refactored ~70 fully qualified class names to proper imports in BossApp.kt
- Removed redundant `_lastInteractedTabPanelId` field from SplitView
- Improved coroutine lifecycle management with rememberCoroutineScope
