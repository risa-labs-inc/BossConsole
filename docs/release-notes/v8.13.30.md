# BOSS v8.13.30

**Release Date**: December 2025

## Summary

This release improves terminal tab management, fixes platform-specific issues on Linux and macOS, and adds configurable title bar settings with platform-specific defaults.

## Highlights

- Terminal tab titles now sync with window title changes
- Fixed Linux dock icon not appearing correctly
- Added configurable title bar with platform-specific defaults
- Fixed menu hang on macOS caused by listener leaks

## Features

### Terminal
- **Sync terminal tab title with window title changes** (#331)
  - Terminal tab titles now automatically update when window title changes
  - Added max length limit of 64 characters for terminal tab titles
  - Prevents excessively long titles from breaking UI layout

### Settings
- **Add configurable title bar with platform-specific defaults** (#327)
  - Users can now toggle between system and integrated title bar
  - Platform-specific defaults: macOS uses integrated, Windows/Linux use system
  - Settings persist across restarts using proper IO dispatchers
  - Includes directory creation for settings storage

## Bug Fixes

### Linux
- **Add StartupWMClass to Linux .desktop file for dock icon** (#329)
  - Fixed dock icon not appearing or appearing as generic icon on Linux
  - Added `StartupWMClass=BOSS` to desktop entry for proper window manager recognition

### macOS
- **Fix listener leaks causing menu hang on macOS** (#319)
  - Fixed issue #296 where menu would hang due to accumulated event listeners
  - Properly cleanup listeners to prevent memory leaks and UI freezes

### CI/CD
- **Rewrite download URLs in release notes to public repo** (#317)
  - Fixed release body download URLs to point to BossConsole-Releases repository
  - Ensures users can access downloads from public release repository
