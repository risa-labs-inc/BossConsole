# BOSS v8.12.6

**Release Date**: October 2024

## Summary

This release removes unused Supabase demo UI components and settings dialogs while preserving all core Supabase functionality for authentication, roles, and database operations.

## Highlights

- Removed Supabase Demo plugin panel that provided basic connection testing interface
- Cleaned up unused settings components (SupabaseSettingsDialog, SupabaseSettings)
- All core Supabase services remain fully functional

## Improvements

### Code Cleanup (#73)

- **Removed Supabase Demo plugin** - Deleted 233-line SupabaseDemoSimple.kt panel component
- **Removed settings dialogs** - Deleted unused SupabaseSettingsDialog.kt (145 lines) and SupabaseSettings.kt (117 lines)
- **Cleaned up dead code** - Removed unused code from BossAppWithAuth.kt and DefaultPlugin.kt
- **Total reduction** - 512 lines of unused code removed

### Core Services Unchanged

All essential Supabase functionality remains intact:
- Authentication services continue working
- Role-based access control (RBAC) fully operational
- Database operations and queries unaffected
- Edge Functions integration preserved
