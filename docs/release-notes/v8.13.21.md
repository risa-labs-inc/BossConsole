# BOSS v8.13.21

**Release Date**: December 2024

## Summary

This release fixes a critical browser stability issue with JxBrowser that could cause crashes due to race conditions in object disposal.

## Highlights

- Fixed JxBrowser 'closed object' crash with thread-safe disposal mechanism
- Improved browser stability during window close and navigation operations

## Bug Fixes

- **Browser**: Resolved JxBrowser 'closed object' crash with thread-safe disposal to prevent race conditions (#255) (#259)
