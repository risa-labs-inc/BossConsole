# BOSS v8.14.10

**Release Date**: December 2025

## Summary

This release improves the new user experience with better project selection workflows, including the ability to skip the project dialog when no recent projects exist and a new "Open Project" button on the dashboard and File tab.

## Highlights

- Project dialog automatically skips when no recent projects exist
- "Open Project" button displayed for new users on dashboard
- Updated plugin icons for better visual identity
- Improved project selection dialog logic and UI consistency

## Features

### Project Management
- **Skip project dialog when no recent projects** (77da93e7)
  - Automatically bypasses project selection dialog for first-time users
  - Streamlines initial setup experience

- **Show Open Project button when no project selected in File tab** (#415)
  - Adds convenience for users to open projects from File tab
  - Better integration with tab-based workflow (f11e6c9f)

### Dashboard
- **Show "Open Project" button for new users** (778f288a)
  - Prominently displays project opening option on dashboard
  - Improves discoverability for first-time users

### CLI
- **Add clear command to workspace CLI templates** (60f67612)
  - Enhanced workspace CLI with clear command support
  - Better terminal integration

## Bug Fixes

### Icons
- **Update plugin icons for better visual identity** (#417)
  - Improved icon consistency and visual design
  - Better UI polish and branding (a9249758)

## Improvements

### Dialog

  - Improved code organization and maintainability
