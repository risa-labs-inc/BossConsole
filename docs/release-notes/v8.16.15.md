# BOSS v8.16.15 Release Notes

**Release Date:** 2026-01-27

## Summary

This release fixes a critical browser focus issue that caused interruptions during split view operations.

## Bug Fixes

### Browser
- **Remove 100ms polling that caused focus stealing in split view** (#568) (ef9ea83)
  - Eliminated aggressive 100ms polling mechanism that was stealing focus from active browser tabs
  - Improves user experience when working with split view configurations
  - Resolves interruptions during typing and interaction in browser panels
