# BOSS v8.17.23

**Release Date:** 2026-03-12

This release is a maintenance update focused on code quality — resolving all remaining compiler warnings across the codebase — alongside five dependency updates including Compose Multiplatform 1.10.2, Ktor 3.4.1, and Netty 4.2.x.

## Highlights

- **Zero compiler warnings**: All 10 outstanding warnings eliminated across `ApplicationEventBus`, `IpcAddressResolver`, `DesktopRpaEngine`, `EditorContentProvider`, and more
- **Compose Multiplatform 1.10.2**: Updated from 1.10.1 with latest upstream fixes
- **Ktor 3.4.1**: Minor update from 3.4.0 with stability improvements
- **Netty 4.2.10.Final**: Major Netty update from 4.1.119.Final on macOS (kqueue transport)

## Improvements

- **build**: Resolve all compiler warnings across codebase — removes 7 unnecessary `UNCHECKED_CAST` suppresses in `ApplicationEventBus`, redundant elvis on non-nullable icon in `TopOfMindDataProvider`, unnecessary casts in `LayoutWorkspace` and `NavigationResolverProviderImpl`, redundant `.toString()` calls in `DesktopRpaEngine`, extracts reusable `Json` instance in `PluginStoreSetup`
- **build**: Replace deprecated `kotlinx.datetime.Clock` with `System.currentTimeMillis()` in affected components
- **build**: Suppress deprecated `LocalClipboardManager` in `CrashReportDialog` and Netty `EventLoopGroup` deprecation in `IpcAddressResolver`
- **build**: Add `@OptIn(DelicateCoroutinesApi)` for `GlobalScope` usage in `EditorContentProvider`

## Dependencies

- **deps**: Bump `compose-multiplatform` from 1.10.1 to 1.10.2
- **deps**: Bump `ktor` from 3.4.0 to 3.4.1
- **deps**: Bump `io.netty:netty-transport-native-kqueue` from 4.1.119.Final to 4.2.10.Final
- **deps**: Bump `com.google.protobuf` from 0.9.5 to 0.9.6
- **deps**: Bump `org.graalvm.buildtools.native` from 0.10.6 to 0.11.5
