# Release Notes - v8.15.24

**Release Date**: January 21, 2026

This release fixes a critical multi-window crash issue and updates the BossTerm terminal integration.

## Highlights

- Fixed crash when creating new browser windows with the correct window handle
- Updated BossTerm dependency to version 1.0.87 for improved terminal functionality

## Bug Fixes

- **browser**: Pass correct window handle to prevent multi-window crash (#501)

## Dependencies

- **terminal**: Update BossTerm to 1.0.87 (#502)
