# BOSS v8.16.24-alpha.1

**Release Date:** 2026-02-06

Alpha release introducing interactive plugin setup wizard for streamlined first-time user onboarding.

## Highlights

- **Plugin Install Wizard**: Multi-step guided wizard for plugin selection during first login
- **Smart Defaults**: Pre-selected essential plugins (Terminal, Console, Codebase, etc.)
- **Category Organization**: Plugins organized by Essential, Developer, Productivity, Automation, Admin, and Other categories
- **Progress Tracking**: Real-time installation progress with completion summary

## Features

- **wizard**: Add plugin install wizard for first-time users (#605)
  - Generic wizard framework with reusable components (WizardState, WizardStep, WizardDialog)
  - Plugin-specific wizard with 6 category steps for easy navigation
  - Pre-selected default plugins to reduce setup friction
  - Installation progress indicators and completion summary
  - Help menu integration for manual access via "Plugin Setup Wizard..."
  - Persistent completion status tracking via UserDataStorage
