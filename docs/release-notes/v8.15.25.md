# Release Notes - v8.15.25

**Release Date**: January 22, 2026

This release focuses on improving multi-window stability by filtering terminal and browser events to their source windows only.

## Highlights

- Fixed runner terminal events incorrectly affecting other windows
- Fixed terminal hyperlink dialogs appearing in wrong windows
- Fixed browser tab title updates not syncing during navigation

## Bug Fixes

- **runner**: Filter runner terminal events to source window only (#498) (#504)
- **browser**: Fix tab title not updating during navigation (#505)
- **terminal**: Filter hyperlink dialog to source window only (#498) (#503)
