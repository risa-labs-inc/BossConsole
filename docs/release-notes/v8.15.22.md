# BOSS v8.15.22

**Release Date**: January 20, 2026

## Summary

This release enhances focus mode with title bar hover reveal functionality, adds search capabilities to BossEditor's large file viewer, and reduces verbose logging for cleaner console output.

## Highlights

- Title bar auto-reveals on hover in focus mode with configurable delay for distraction-free workflows (#494)
- Search functionality with case sensitivity toggle added to LargeFileViewer for better navigation in large files (#492)
- Reduced verbose logging from WindowFocusManager and HQFavicon for cleaner application logs (#493)

## Features

### Focus Mode
- **Title bar hover reveal** (#494)
  - Added automatic title bar reveal when hovering near the top of the window in focus mode
  - Configurable hover delay to prevent accidental reveals during normal navigation
  - Enhances distraction-free workflow while maintaining quick access to window controls

### BossEditor
- **Large file viewer search** (#492)
  - Added search functionality to LargeFileViewer component
  - Case sensitivity toggle for flexible search options
  - Enables efficient navigation within memory-efficient large file viewing

## Improvements

### Logging
- **Reduced verbose logging** (#493)
  - Removed excessive debug output from WindowFocusManager
  - Cleaned up verbose logging in HQFavicon component
  - Improves console readability during normal application usage
