# BOSS v8.13.36

**Release Date**: December 2025

## Summary

This release introduces comprehensive drag-and-drop support for tabs, enabling users to reorder tabs within windows and move tabs between windows through intuitive drag gestures.

## Highlights

- Added full drag-and-drop support for tab reordering and cross-window moves
- Fixed "Open in New Window" context menu to properly move tabs to new windows
- Improved tab drag cleanup and lifecycle management
- Enhanced drag-drop safety and performance with better error handling
- Updated BossTerm to 1.0.57

## Features

### Tabs
- **Add drag-and-drop support for tabs**
  - Enabled dragging tabs within the same window to reorder
  - Enabled dragging tabs between windows to move them
  - Implemented visual drag ghost and drop indicators
  - Added support for drag-and-drop with external window management

## Bug Fixes

### Tabs
- **Open in New Window context menu now properly moves tab to new window**
  - Fixed context menu action to actually move the tab instead of just opening a new window
  - Ensures tab is removed from source window and added to destination window

- **Preserve current URL when splitting browser tabs** (#282)
  - Fixed bug where current URL was lost when splitting browser tabs
  - Ensures new split tab starts with the same URL as the source tab

- **Use correct tab count accessor in handleTabDropResult**
  - Fixed tab count calculation to use proper accessor method
  - Prevents index out of bounds errors during drag-drop operations

- **Address critical issues in tab drag-drop operations**
  - Fixed race conditions in drag state management
  - Added validation for tab indices during drop operations
  - Improved error handling for edge cases

- **Improve drag cleanup and Fluck tab lifecycle management**
  - Enhanced cleanup logic to prevent orphaned drag states
  - Fixed Fluck tab lifecycle issues during drag operations
  - Ensures proper disposal of resources after drag completion

## Improvements

### Tabs
- **Improve drag-drop safety and performance**
  - Added bounds checking for tab operations
  - Optimized drag gesture detection
  - Enhanced performance during rapid drag operations
  - Added defensive null checks and error recovery

## Dependencies

- **Update BossTerm to 1.0.57**
  - Latest terminal component improvements and bug fixes
