# BOSS v8.14.9

**Release Date**: December 2025

## Summary

This release introduces a new Dashboard feature for quick access to recent items and workspace templates, along with important performance optimizations and memory leak fixes.

## Highlights

- New Dashboard with recent items, templates, and workspace settings
- Fixed HTTP client memory leak on shutdown
- Optimized dashboard file I/O and JSON parsing performance
- Improved Window menu split functionality and new tab dialog usability

## Features

### Dashboard
- **Add Dashboard with recent items, templates, and workspace settings** (d6d6e278)
  - Quick access to recently opened projects and files
  - Workspace templates for faster project setup
  - Centralized workspace configuration management

### Split View
- **Disable split menu options when no tabs are open** (0d9f52e6)
  - Prevents confusion when split operations aren't available
  - Better UI state consistency

### Menu
- **Move New Window option to File menu** (30326c01)
  - Improved menu organization for better discoverability

## Bug Fixes

### Memory
- **Close HTTP client on shutdown to prevent memory leak** (882db8e6)
  - Fixed resource cleanup issue causing memory retention
  - Improves application stability on exit

### New Tab Dialog
- **Fix folder expansion and update URL wording** (fa6aae27)
  - Resolved folder expansion UI issues
  - Clearer URL field labeling

### Split View
- **Fix Window menu split glitch by copying active tab** (b89c06bb)
  - Corrected split view behavior in Window menu
  - Ensures active tab is properly duplicated during split

### Dashboard
- **Address code review threading, memory, and performance issues** (e0e668db)
  - Fixed threading issues in dashboard components
  - Improved memory management
  - Enhanced overall dashboard performance

## Improvements

### Performance
- **Optimize file I/O, JSON parsing, and favicon fetching** (b8dcc9ad)
  - Faster dashboard loading and rendering
  - Reduced JSON parsing overhead
  - More efficient favicon retrieval

## Dependencies

- **Update BossTerm to 1.0.70** (c135d261)
  - Latest terminal integration improvements
- **Update BossTerm to 1.0.69** (c3223f38)
  - Terminal bug fixes and enhancements
