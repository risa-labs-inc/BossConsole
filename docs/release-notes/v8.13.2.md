# BOSS v8.13.2

**Release Date**: November 2025

## Summary

This release implements intelligent popup handling for browser links, fixing OAuth authentication flows and preventing mail app links from opening in detached OS windows.

## Highlights

- Smart popup detection distinguishes OAuth popups from regular links (#195)
- OAuth authentication flows now work correctly with Google, GitHub, etc. (#173)
- Mail app and target="_blank" links open as tabs instead of OS windows (#137)
- Proper Swing window management for OAuth popups with lifecycle handling

## Bug Fixes

- **Browser**: Intelligent popup handling for OAuth and mail links (Issues #137, #173) (#195)
  - Implemented popup callback with bounds checking to distinguish link types
  - Links without window dimensions (mail links, target="_blank") now open as tabs in BOSS
  - Links with specific dimensions (OAuth popups) open as proper popup windows
  - Fixed OAuth flows to maintain window.opener communication channel
  - Added navigation event listener for empty URL handling in mail links
  - Fixed memory leaks by properly unsubscribing event listeners with timeout mechanism
  - Eliminated race condition using AtomicBoolean for cleanup coordination
  - Prevented coroutine scope leak by cancelling timeout handler on successful navigation

## Improvements

- **Browser**: Enhanced popup window management
  - Added Swing JFrame implementation for OAuth popup windows
  - Dynamic title updates from page title
  - Proper window positioning and sizing from initial bounds
  - Bidirectional close event handling between browser and window

## Dependencies

- Added `jxbrowser.swing` dependency for popup window support
