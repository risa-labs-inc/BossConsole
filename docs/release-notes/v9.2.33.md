# BOSS v9.2.33

**Release Date:** 2026-07-06

Maintenance release with no functional code changes — a version bump on top of v9.2.32 to publish a fresh signed multi-platform build.

## Highlights

- **No functional changes**: The only in-range commit is documentation (the v9.2.32 release notes); all v9.2.32 functionality carries forward unchanged.
- **Carries forward the macOS launch fix**: v9.2.32's removal of the still-ungranted `com.apple.developer.web-browser.public-key-credential` managed-capability entitlement from `BOSS.entitlements` — which had blocked every macOS launch of the signed app — remains in effect (#836).
