# BOSS v8.17.4

**Release Date:** 2026-02-19

Adds browser find-in-page (Cmd+F) and paste-without-formatting (Cmd+Shift+V), Supabase Realtime support for live plugin store updates, and a HYBRID plugin type fix, alongside dependency updates for Compose Multiplatform 1.10.1 and BossTerm 1.0.90.

## Highlights

- **Browser Find-in-Page**: Cmd+F now opens a native-styled find bar in browser tabs using JxBrowser TextFinder API with match counter, case toggle, and navigation buttons
- **Paste Without Formatting**: Cmd+Shift+V strips rich formatting when pasting in browser tabs, compatible with Google Docs and all web editors
- **Supabase Realtime for Plugins**: Plugin store tables now publish live updates so the Plugin Manager can receive real-time changes without polling
- **HYBRID Plugin Type Fix**: Resolves a `SerializationException` crash when reading manifests for hybrid-type plugins

## Features

- **plugins**: Enable Supabase Realtime on `plugins` and `plugin_versions` tables for live Plugin Manager updates

## Bug Fixes

- **browser**: Add Cmd+F find-in-page using a Swing-based find bar (undecorated JDialog) with dark theme, match counter, case toggle, 150ms search debounce, and proper resource cleanup on browser close (#597) (#636)
- **browser**: Add Cmd+Shift+V paste-without-formatting via clipboard replacement and native Cmd+V dispatch through JxBrowser browser.dispatch() API (#636)
- **browser**: Fix context-aware shortcut matching in AWTKeyboardInterceptor to prevent cross-context shortcut conflicts (editor/browser/terminal) (#636)
- **plugins**: Add `HYBRID` to `PluginType` enum to fix `SerializationException` when reading manifests from `boss-plugin-api-1.0.23.jar`

## Dependencies

- Compose Multiplatform 1.10.0 → 1.10.1 (#632)
- `com.risaboss:bossterm-compose` 1.0.89 → 1.0.90 (#633)
- `ch.qos.logback:logback-classic` 1.5.28 → 1.5.31 (#634)
- `org.junit.jupiter:junit-jupiter` 5.10.2 → 6.0.3 (#635)
