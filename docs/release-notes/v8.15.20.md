# BOSS v8.15.20

**Release Date**: January 19, 2026

## Summary

This release fixes a critical CI/CD workflow regression that prevented proper multi-platform builds and includes GitHub Actions dependency updates.

## Highlights

- Restored all 5 platform build configurations in release workflow that were accidentally removed during merge (#485)
- Updated actions/checkout from v5 to v6 for improved CI/CD reliability

## Bug Fixes

### Workflow
- **Restore 5 build jobs in release workflow** (#485)
  - Fixed accidental revert of release.yml to older version during dependabot merge
  - Restored all 5 platform build configurations: macOS ARM64, Linux AMD64, Linux ARM64, Windows x64, and Windows ARM64
  - Ensures proper multi-platform distribution for all supported architectures

## Dependencies

- Updated actions/checkout from 5 to 6
