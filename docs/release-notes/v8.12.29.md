# BOSS v8.12.29 Release Notes

Released: December 2025

## Summary

This release improves form autofill compatibility by ensuring dynamic content changes work correctly with React 16+ applications.

## Highlights

- Enhanced form autofill system to support React 16+ with proper event dispatching (#143)
- Fixed autofill compatibility issues with modern JavaScript frameworks

## Bug Fixes

### Browser
- **Form Autofill Compatibility**: Enhanced form autofill to properly trigger React and other framework event handlers by dispatching both 'input' and 'change' events with proper configuration (bubbles, composed, cancelable) (#143)
- **React 16+ Support**: Fixed autofill values not being recognized by React 16+ applications by ensuring events are dispatched in correct order and using proper event types (#143)
