# BOSS v8.13.5

**Release Date**: November 2025

## Summary

This release fixes a critical usability issue where keyboard shortcuts would incorrectly trigger while typing in browser input fields, interrupting user text entry.

## Highlights

- Fixed keyboard shortcuts triggering during browser text input (#214)

## Bug Fixes

- **Keyboard Shortcuts**: Fixed keyboard shortcuts no longer triggering while typing in browser input fields (#214)
  - Shortcuts now properly respect input field focus state
  - Prevents accidental command execution during text entry
  - Improves overall browser usability and user experience
