# BOSS v8.12.2

**Release Date**: October 2024

## Summary

This release improves passkey authentication with proper rpId configuration and browser compatibility, adds magic link deep linking support, and includes several dependency updates.

## Highlights

- Fixed passkey authentication with proper rpId configuration for WebAuthn (#64)
- Added redirect Edge Function for seamless magic link deep linking (#52)
- Updated GitHub issue templates with consistent formatting (#39)
- Multiple dependency updates for improved stability

## Features

### Magic Link Deep Linking (#52)

**Redirect Edge Function:**
- Added new Supabase Edge Function for handling magic link redirects
- Enables seamless deep linking from email to BOSS application
- Configured email template for proper deep link handling
- Updated Supabase Cloud configuration for magic link support

## Bug Fixes

### Passkey Authentication Fix (#64)

**rpId Configuration:**
- Fixed passkey authentication with proper rpId (Relying Party ID) configuration
- Resolved WebAuthn browser compatibility issues
- Improved passkey registration and authentication flow reliability

**Configuration Updates:**
- Corrected Supabase function URL path
- Clarified Supabase Cloud configuration for magic links

## Improvements

### GitHub Templates (#39)

- Updated issue templates with consistent formatting
- Improved contributor experience for bug reports and feature requests

### Manual Cleanup (#63)

- Code cleanup and maintenance improvements
- Removed unused code and improved code quality

## Dependencies

- Updated `supabase` from 3.2.2 to 3.2.4 (#62)
- Updated `decompose` from 3.3.0 to 3.4.0 (#60)
- Updated `androidx-lifecycle` from 2.9.3 to 2.9.4 (#59)
- Updated `ch.qos.logback:logback-classic` (#58)
- Updated `net.java.dev.jna:jna-platform` (#61)
