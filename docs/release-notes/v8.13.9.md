# BOSS v8.13.9

**Release Date**: November 2025

## Summary

This release improves CI/CD reliability by preventing race conditions and consolidating the release workflow to eliminate duplicate workflow runs.

## Highlights

- Fixed race condition in release workflow by tagging specific commit SHA
- Consolidated release workflow to prevent duplicate runs (#218)

## Bug Fixes

- **Workflow**: Prevent race condition by tagging specific commit SHA
  - Ensures consistent tagging by targeting the exact commit SHA
  - Eliminates timing issues in release automation

- **Workflow**: Consolidate release workflow to prevent duplicate runs (#218)
  - Merged separate workflow triggers into single unified workflow
  - Prevents multiple simultaneous release builds from different trigger sources
  - Improves reliability and reduces confusion from parallel release attempts
