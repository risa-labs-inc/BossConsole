# BOSS v8.12.14

**Release Date**: October 2024

## Summary

This release fixes embedded browser functionality for Settings passkey registration and adds auto-click capability to streamline the registration process.

## Highlights

- Fixed embedded browser integration for passkey registration in Settings
- Added auto-click functionality to improve registration flow
- Resolved Issue #79 for seamless passkey setup

## Bug Fixes

### Authentication
- **Fixed embedded browser for Settings passkey registration** (#102, Issue #79)
  - Resolved browser embedding issues preventing passkey registration from Settings
  - Added auto-click capability to streamline the registration workflow
  - Improved user experience during passkey setup process
