# BOSS v8.12.32 Release Notes

Released: December 2025

## Summary

This release focuses on browser usability improvements with enhanced context menu functionality, better URL bar interaction, and improved window focus handling for deep link navigation.

## Highlights

- Added copy text option to browser context menu for easier text selection (#161)
- Fixed window focus issues when handling deep links on startup
- Improved URL bar cursor visibility and navigation experience
- Enhanced New Tab Dialog with arrow key support for URL suggestions

## Bug Fixes

### Browser

- **URL Bar**: Show cursor in empty URL bar by always rendering innerTextField (#157)
- **Navigation**: Enable arrow key navigation in New Tab Dialog URL suggestions (#156)

### Features
