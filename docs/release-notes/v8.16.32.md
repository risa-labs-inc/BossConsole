# BOSS v8.16.32

**Release Date:** 2026-02-09

Build system reliability improvements with task dependency fixes and Kotlin platform updates.

## Highlights

- **Build Task Dependencies**: Fixed prepareAppResources task ordering to prevent build failures
- **Kotlin Update**: Updated to Kotlin 2.3.10 for latest compiler improvements and bug fixes
- **Logging Framework**: Updated Logback to 1.5.18 for improved logging stability

## Bug Fixes

- **build**: Add task dependency for prepareAppResources (ddda5d6a)
  - Fixed task ordering issue where prepareAppResources could execute before required dependencies
  - Ensures proper resource preparation during build process
  - Prevents intermittent build failures

## Dependencies

- **kotlin**: Bump kotlin from 2.3.0 to 2.3.10 (#620)
  - Compiler improvements and bug fixes
  - Enhanced language features stability
  - Performance optimizations

- **logback**: Bump ch.qos.logback:logback-classic (#619)
  - Updated to version 1.5.18
  - Improved logging framework stability
  - Enhanced thread safety and performance

## Improvements

- **plugin-api**: Add PluginLoaderDelegate and getPluginAPI infrastructure (#618)
  - New PluginLoaderDelegate interface for plugin API access
  - Added getPluginAPI() method to PluginLoader for retrieving plugin API instances
  - Enables plugins to dynamically access BOSS plugin APIs at runtime
  - Foundation for more flexible plugin architecture
