# BOSS v8.16.27-alpha.1

**Release Date:** 2026-02-07

Major plugin API enhancements for dynamic tab providers and improved browser URL handling for better navigation experience.

## Highlights

- **Dynamic Tab Providers**: New content providers for Terminal and Editor tabs enable dynamic plugin-generated tabs
- **Enhanced Plugin API**: Browser feature parity and comprehensive provider APIs for dynamic plugins
- **Browser Navigation Fix**: URL bar now only updates for main frame navigations, preventing unwanted updates from iframes
- **Plugin Wizard UX**: Window close button now enabled in the plugin setup wizard

## Features

- **plugin-api**: Add TerminalTabContentProvider and EditorContentProvider for dynamic tab plugins
  - Enable plugins to dynamically create terminal and editor tabs
  - Provide content provider interfaces for plugin-generated tab content
  - Support dynamic tab lifecycle management through plugin APIs

- **plugin-api**: Add providers for dynamic editor plugin parity
  - Ensure editor plugins have equivalent capabilities as terminal plugins
  - Provide consistent provider APIs across tab types

- **plugin-api**: Add TerminalTabInfoInterface for dynamic terminal plugins
  - Define interface for terminal tab information and metadata
  - Enable plugins to query and manage terminal tab state

- **plugin-api**: Add browser feature parity and new providers for Fluck plugin (#610)
  - Expand browser integration capabilities for plugins
  - Add specialized providers for advanced plugin use cases
  - Improve plugin ecosystem support with enhanced APIs

## Bug Fixes

- **browser**: Only update URL bar for main frame navigations (28a76da4)
  - Fix URL bar incorrectly updating when iframes navigate
  - Ensure URL bar only reflects actual page navigations
  - Improve browser address bar accuracy and user experience

- **plugin-manager**: Add apiVersion, minBossVersion, and type to manifest extraction (f6b247a3)
  - Extract critical metadata fields from plugin manifests
  - Support version compatibility checking between plugins and BOSS
  - Enable proper plugin type identification and validation

- **wizard**: Enable window close button in plugin wizard (#609)
  - Allow users to close the plugin setup wizard window
  - Improve wizard window behavior and user control

## Dependencies

- **plugin-api**: Bump version to 1.0.12
