# BOSS v9.2.26

**Release Date:** 2026-07-03

Adds a public, key-free `latest-release` edge function so the project's Downloads table can link straight to the newest installer per platform/arch instead of the releases browse page.

## Highlights

- **Stable "download latest" links**: A new public `latest-release` edge function serves both the latest release JSON and stable 302 redirects to the newest installer per platform, so download links no longer need to be re-pointed at each versioned asset (#822).
- **Always-latest README Downloads table**: The README Downloads table now links directly to the newest installer per platform/arch (split into per-arch rows), with the releases page kept as the browse-all-versions fallback (#824).

## Features

- **release**: Add a public, key-free `latest-release` edge function (deployed with `verify_jwt=false`) over the `app_releases` catalog. `GET /latest-release?app=boss|bossterm` returns the latest release JSON, and `?download=dmg|msi|...` issues a `302` to the newest installer (optional `arch=arm64|amd64` with `aarch64`/`x86_64`/`x64` aliases, plus `channel=` and `prerelease=true`). It reads the anon-readable table with the runtime anon key so browser/website links don't need an `apikey`, keeps the versioned filename on redirect, and sets `Cache-Control: no-store` so a cached `302` can't serve a stale installer. The asset picker handles the catalog's naming quirks: it requires the release version in the filename (excluding SDK jars), treats `Universal.dmg` as satisfying both Mac arches, prefers the un-suffixed `BOSS-x.y.z.msi` when no arch is given, and defaults multi-arch Linux to amd64. Backed by 14 deno tests (#822).

## Bug Fixes

- **release**: Arch-filtered downloads now degrade exact-arch → universal → un-suffixed default installer before returning `404`. `download=msi&arch=amd64` previously 404ed because BOSS ships the default Windows build as `BOSS-x.y.z.msi` (no arch token) alongside `-arm64.msi`, and the arch branch only fell back to Universal builds; the same failure hit `app=bossterm&download=dmg&arch=<anything>` for BossTerm's single arch-less dmg (#822).

## Improvements

- **docs**: Point the README Downloads table at the always-latest links exposed by the new edge function, split into per-arch rows (Windows x64 vs ARM64, Linux AMD64 vs ARM64). All nine links were verified live against v9.2.25, including the plain-msi default for x64 and the arch-tagged variants; the releases page is retained as the browse-all-versions fallback and a JSON metadata link is included for scripts (#824).
