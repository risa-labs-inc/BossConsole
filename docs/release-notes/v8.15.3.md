# BOSS v8.15.3

**Release Date**: January 5, 2026

## Summary

This release fixes a critical browser bug related to URL recovery using stale object references.

## Highlights

- Fixed browser URL recovery crash caused by stale object reference (#434)

## Bug Fixes

### Browser
- **Resolve URL recovery using stale object reference** (#434)
  - Fixed crash when attempting to recover browser URL after navigation failures
  - Ensures proper object lifecycle management in browser state recovery (9f337ea7)
