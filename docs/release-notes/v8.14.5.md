# BOSS v8.14.5

**Release Date**: December 2025

## Summary

This release brings significant improvements to BossEditor with enhanced navigation feedback, performance optimizations for large documents, comprehensive error handling, and expanded test coverage. Also includes CI/CD dependency updates and terminal integration improvements.

## Highlights

- New feedback popup when code navigation targets are not found in BossEditor
- Major performance optimization for large documents with incremental line index updates
- Comprehensive error handling improvements in document operations
- Expanded test coverage with syntax highlighting verification and benchmarks
- Updated GitHub Actions workflow dependencies and BossTerm integration

## Features

### BossEditor
- **Add feedback popup when navigation target not found** (#388)
  - Displays user-friendly popup when Go to Definition or similar navigation fails
  - Improves developer experience by providing clear feedback
  - Addresses code review feedback on NavigationFeedbackPopup implementation

## Bug Fixes

### BossEditor
- **Improve error handling in document operations** (#391)
  - Enhanced error handling across document manipulation operations
  - Better recovery from edge cases and invalid states
  - Addressed review feedback on error handling patterns
- **Add coroutineScope to handleNavigationRequest remember keys**
  - Fixed potential coroutine lifecycle issues in navigation handling
  - Ensures proper cleanup of navigation coroutines

## Improvements

### BossEditor
- **Optimize large document performance with incremental line index updates** (#390)
  - Significant performance improvement for documents with thousands of lines
  - Incremental updates instead of full reindexing
  - Reduced memory allocations during editing operations

### Testing
- **Add comprehensive test coverage for core components** (#389)
  - Expanded unit tests for BossEditor core functionality
  - Improved code reliability and maintainability
- **Add syntax highlighting verification and rendering benchmarks** (#392)
  - Automated verification of syntax highlighting correctness
  - Performance benchmarks for rendering operations
  - Ensures consistent highlighting quality across updates

## Dependencies

- **Update BossTerm to 1.0.65** (4764aacc)
  - Latest terminal integration improvements and fixes
- **Bump actions/upload-artifact from 5 to 6** (#386)
  - Updated GitHub Actions workflow dependency
- **Bump actions/download-artifact from 6 to 7** (#387)
  - Updated GitHub Actions workflow dependency
