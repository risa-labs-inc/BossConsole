# BOSS v8.17.13

**Release Date:** 2026-03-07

Plugin API expansion release adding a generic `SupabaseDataProvider` interface that allows plugins to execute Postgrest queries without importing the Supabase SDK directly.

## Highlights

- **SupabaseDataProvider interface**: Plugins can now query Supabase via `select()` and `rpc()` methods using plain JSON strings, with no Supabase SDK dependency required in plugin code
- **Wired through plugin context**: Implementation flows through `DefaultPlugin` and `TrackingPluginContext`, making the provider automatically available to all plugins via `PluginContext`

## Features

- **plugin-api**: Add `SupabaseDataProvider` interface with `select()` and `rpc()` methods for Postgrest queries from plugins
- **plugin-api**: Expose `SupabaseDataProvider` on `PluginContext` so plugins can access Supabase data without SDK imports
- **plugin**: Wire `SupabaseDataProviderImpl` through `DefaultPlugin` and `TrackingPluginContext`
