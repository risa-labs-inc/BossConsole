# BOSS v9.1.3

**Release Date:** 2026-05-09

Maintenance release that keeps the top-bar git state in sync when the repository's HEAD is changed outside BOSS (CLI checkout, rebase, etc.) and stops the plugin install wizard from attempting to install the microkernel runtime as a regular plugin.

## Highlights

- **Live top-bar git refresh on external HEAD changes**: The top-bar branch label, file status, stash list, and commit log now refresh automatically when `git checkout`, `git switch`, or a rebase mutates `.git/HEAD` or `.git/refs/heads` outside the app, via a coalesced NIO `WatchService` keyed to the active project.
- **Wizard no longer tries to install the microkernel runtime**: Two-layer guard so the runtime no longer surfaces in the install wizard UI or fails with a cross-classloader `kotlin.reflect` `IllegalAccessError` when the bulk-install path bypasses the plugin-store delegate.

## Features

- **git**: Add `watchGitHeadForWindow(projectPath, windowGitState)` to the `GitService` expect/actual surface; desktop impl watches `.git` (HEAD / packed-refs) and `.git/refs/heads` (branch tip rewrites) with a 150 ms coalesce window, refreshing status, stash list, and commit log on every relevant event (#699)

## Bug Fixes

- **plugins**: Stop the plugin install wizard from offering or installing the microkernel runtime — `PluginListProvider` now drops `PluginType.SERVICE` plugins and the runtime `pluginId`, and `PluginInstallService.installPlugins` short-circuits the runtime as already-installed as defense-in-depth so it never reaches the cross-classloader `kotlin.reflect.TypeVariableImpl` failure path
- **git**: Refresh the commit log alongside branch label and file status on external HEAD changes so the git-log plugin no longer shows stale history after a checkout (#699)

## Improvements

- **ui**: `BossTopBar` launches the git watcher in a sibling `LaunchedEffect` keyed on `projectPath`/`windowGitState`, so the watcher lifetime tracks the UI's interest in the repo and is auto-cancelled when the project changes or composition leaves the tree (#699)
